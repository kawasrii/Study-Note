タグ: 語彙 / フラッシュカード / 4択クイズ / 学習記録(このブラウザ内のみ保存)

全体マップ — 語彙学習の体系の中での位置づけ

  • 語彙力
    • 広さ — 知っている語の数
      • 直接暗記: コア語彙100(このページ) — 即効性が高い
      • 推測力: 語根・接頭辞 — 覚えた量を超えて効く保険
    • 深さ — 正しく使い分けられる度合い
      • 使い分け: 紛らわしい語ペア — 準備中
      • 組み合わせ: コロケーション — 準備中

「広さ」の2本(暗記と推測)を並行で回し、余裕が出たら「深さ」に進むのが効率的。

このページは①を担当する。読解でもリスニングでも頻出する上級レベルの基礎100語。おすすめの回し方:①フラッシュカードを1周(わからなくても止まらない)→ ②クイズで弱点を炙り出す → ③「まだ」の単語だけもう1周。例文ごと音読すると定着が段違いに良くなります。

4択クイズ

英単語 → 意味を選ぶ(出題はシャッフル)
🔧 実装メモ(このページの仕組み)
  • 収録語の選定: 上級レベル(英検準1級・TOEIC 700〜800点帯相当)の頻出基礎語からアカデミック・ビジネス両方で使う100語を採録。特定試験の公式リストには準拠していない
  • シャッフル: Fisher–Yates法(偏りのない一様シャッフル)
  • クイズ: ページ読込ごとに100語から20問を無作為抽出。誤答選択肢は他の99語の意味から無作為に3つ(意味の類似度は考慮していないため、たまに簡単な問題が出る)
  • 学習記録: 「覚えた」単語は localStorage(キー: sa_vocab_core_known)にこのブラウザ内にのみ保存。外部送信なし。端末・ブラウザを変えると記録は引き継がれない
  • カードめくりは CSS 3D transform。間隔反復(SRS)アルゴリズムは未実装 — 「覚えた/まだ」の2値管理のみ
