紛らわしい語ペア:似た顔の単語を見分ける

タグ: 語彙 / 使い分け / affect・effect / imply・infer / 混同しやすい単語

全体マップ — 語彙学習の体系の中での位置づけ

  • 語彙力
    • 広さ — 知っている語の数
      • 直接暗記: コア語彙100
      • 推測力: 語根・接頭辞
    • 深さ — 正しく使い分けられる度合い
      • 使い分け: 紛らわしい語ペア(このページ)
      • 組み合わせ: コロケーション

「知っている」のに読み間違える・書き間違える原因の多くがこのページの語たち。品詞の違い・視点の違い(与える側か受ける側か)・語源の違いのどれで区別できるかを意識すると、丸暗記より圧倒的に定着します。多くは語根・接頭辞の知識で説明がつきます。

頻出ペア一覧 — 見分けのコツつき

ペア意味の違い見分けのコツ例文
affect / effectaffect=動詞「影響する」/ effect=名詞「効果」Action(動詞)のaffect、End result のeffectSmoking affects health. / The effect was huge.
adapt / adoptadapt=適応する・改作する / adopt=採用する・養子にするadapt=adjust、adopt=chooseAdapt to change. / Adopt a new policy.
imply / inferimply=(話し手が)ほのめかす / infer=(聞き手が)推論するin(中へ)+ply=折り込む / in+fer=運び込む=読み取る。矢印の向きが逆He implied it. / I inferred it from his tone.
lie / laylie=横たわる(自動詞: lie-lay-lain)/ lay=横たえる(他動詞: lay-laid-laid)layは目的語が要る。lieの過去形がlayなのが混乱の元I lay down yesterday. / Lay the book down.
rise / raiserise=上がる(自動詞)/ raise=上げる(他動詞)raiseは目的語が要る(lie/layと同型)Prices rise. / They raised prices.
borrow / lendborrow=借りる / lend=貸す矢印の向き: borrow=自分へ、lend=相手へMay I borrow a pen? / I'll lend you one.
economic / economicaleconomic=経済の / economical=節約になる-al が付くと日常的な「おトク」の意味にeconomic growth / an economical car
sensible / sensitivesensible=分別のある / sensitive=敏感なsensible=常識(sense)がある、sensitive=感じ(sense)やすいa sensible decision / sensitive skin
principal / principleprincipal=主要な・校長 / principle=原理・主義校長先生は my pal(語尾で覚える定番)the principal reason / moral principles
stationary / stationerystationary=静止した / stationery=文房具stationery の e は envelope(封筒)a stationary car / buy stationery
complement / complimentcomplement=補完する / compliment=ほめ言葉complement=complete(完全にする)Wine complements cheese. / a nice compliment
ensure / assure / insureensure=確実にする / assure=(人に)保証する / insure=保険をかけるassureの目的語は人、insureはお金の話Ensure safety. / I assure you. / Insure the car.
accept / exceptaccept=受け入れる / except=〜を除いてex(外へ)+cept=外に取り出す=除外Accept the offer. / Everyone except me.
later / latterlater=あとで(時間)/ latter=後者(対比)latterは the latter で「後者」。former(前者)とペアSee you later. / The latter option is better.
自動詞/他動詞ペア(lie/lay, rise/raise)は「目的語があるか」だけで機械的に判定できる。imply/infer は「発信者の動詞か受信者の動詞か」— 統計の文章では "the data imply/we infer" の形で頻出し、推測統計の議論そのものでもある。
