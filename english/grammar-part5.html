Part5型文法演習:まず「問題の型」を見抜く

タグ: TOEIC Part5 / 文法 / 品詞問題 / 前置詞 / 接続詞 / 解法ストラテジー

全体マップ — 空所補充は4つの型しかない

なぜ型なのか: Part5(短文穴埋め30問)を1問20秒で処理するには、全文を読む前に「何を試されているか」を判定するのが先。選択肢を見れば型は数秒で分かり、型ごとに見る場所が決まっています。950点レベルとは「型判定の自動化」のこと。

  • Part5の設問タイプ
    • ① 品詞問題 — 選択肢が同語源の変化形(economy/economic/economically…)
      • 見る場所: 空所の前後だけ。冠詞と名詞の間→形容詞、動詞の後→副詞、が典型
    • ② 動詞の形 — 時制・態・主述一致(has done / was done / doing…)
      • 見る場所: 主語との一致 → 能動/受動(目的語の有無)→ 時制のヒント語(since, by the time)
    • ③ 前置詞 vs 接続詞 — despite/although, during/while, because of/because
      • 見る場所: 空所の後ろが名詞句なら前置詞、主語+動詞(節)なら接続詞
    • ④ 語彙(意味)問題 — 型では解けない。語彙力とコロケーションの出番
      • 見る場所: 文全体。①〜③を秒殺して、ここに時間を回すのが時間戦略
型選択肢の見た目判定のカギ目標時間
① 品詞-tion / -al / -ly / 動詞形が並ぶ空所の前後の品詞スロット10秒
② 動詞の形同じ動詞の活用が並ぶ主語の数・目的語の有無・時のヒント語15秒
③ 前置詞/接続詞despite, although, during…が混在後ろが名詞句か節か10秒
④ 語彙意味の違う単語が並ぶ文脈+コロケーション20〜30秒

型別ミニ講義

① 品詞: 英文は「スロット」の言語。the ___ growth なら冠詞と名詞に挟まれた空所=形容詞(economic)。grew ___ なら動詞を修飾=副詞(rapidly)。意味を考える前に構造で確定する。

② 動詞の形: 手順は固定 — (1) 主語は単数か複数か (2) 空所の後に目的語があるか(なければ受動態の可能性大) (3) since+過去なら現在完了、by the time+過去なら過去完了。

③ 前置詞/接続詞: ___ the heavy rain, the game continued. 後ろは the heavy rain(名詞句)→ 前置詞 Despite。___ it rained heavily, ... 後ろは節 → 接続詞 Although。この1点だけで即断できる。
