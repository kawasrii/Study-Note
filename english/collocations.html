コロケーション:自然な語の組み合わせ

タグ: コロケーション / make・do・take・have / 動詞+名詞 / 形容詞+名詞 / 表現力

全体マップ — 語彙学習の体系の中での位置づけ

  • 語彙力
    • 広さ — 知っている語の数
      • 直接暗記: コア語彙100
      • 推測力: 語根・接頭辞
    • 深さ — 正しく使い分けられる度合い
      • 使い分け: 紛らわしい語ペア
      • 組み合わせ: コロケーション(このページ)

なぜ学ぶのか: 単語を知っていても "do a mistake"(正しくは make a mistake)のように組み合わせが不自然だと一発で非ネイティブの英語になる。コロケーションは文法規則では説明できない「慣習の相性」で、丸ごと覚えるしかない — 逆に言えば、頻出の組み合わせを塊で覚えるだけで、作文・会話の自然さが一段上がる即効性の高い分野。

最重要: make / do / take / have の使い分け

動詞イメージ定番の組み合わせ
make無から生み出す・決めるmake a decision / a mistake / an effort / progress / a promise / money / a difference
do作業・義務をこなすdo homework / research / business / the dishes / one's best / harm / damage
take取り込む・引き受けるtake a break / a risk / responsibility / notes / a picture / action / part in
have経験・時間を持つhave a look / a rest / breakfast / a meeting / fun / trouble / an effect on
迷ったら: 成果物や決定を「生む」なら make、作業を「こなす」なら do。make progress(進歩を生む)と do research(研究作業をこなす)の対比が典型。「努力する」は make an effort と do one's best の両方があるが動詞は交換不可 — 塊で覚える理由がここにある。

その他の頻出コロケーション

型自然な組み合わせ ○ありがちな誤り ✗
形容詞+名詞heavy rain / strong coffee / high temperaturestrong rain ✗ / heavy coffee ✗ / tall temperature ✗
形容詞+名詞heavy traffic / a strong possibility / fast foodcrowded traffic ✗ / quick food ✗
副詞+形容詞highly unlikely / deeply concerned / fully awarevery unlikely(可だが弱い)/ strongly concerned ✗
動詞+名詞pay attention / keep a promise / break a record / catch a coldgive attention ✗ / hold a promise ✗
動詞+名詞raise a question / draw a conclusion / meet a deadlinelift a question ✗ / take a conclusion ✗
統計・学術系conduct research / draw a conclusion / test a hypothesis / a significant differencemake research ✗ / do a conclusion ✗

最後の行は準1級の英語論文読解でもそのまま役立つ学術コロケーション。仮説検定の英語表現(test a hypothesis, reject the null hypothesis, statistically significant)は塊で覚えておくと洋書・論文がぐっと読みやすくなります。
