推定:最尤法・推定量の性質・区間推定

タグ: 最尤法 / モーメント法 / 不偏性 / 一致性 / 有効性 / フィッシャー情報量 / クラメール・ラオの不等式 / 十分統計量 / 漸近正規性 / 区間推定

フェーズ2対応済み: 対数尤度カーブのラボ・正規MLEの導出・バイアスの証明つき。

前提知識: 標本平均と母平均、不偏分散(n−1)の区別は記述統計の基礎、期待値・分散の定義は確率変数と期待値・分散の定義にあります。

全体マップ — この分野の体系

推定の問いは2つ:「どう作るか(推定量の構成法)」と「どれが良いか(推定量の評価基準)」。準1級最頻出の最尤法は構成法の王様で、その良さ(漸近有効性)は評価理論(クラメール・ラオ)で保証される — 両者が噛み合う構造を掴むこと。

      • モーメント法 — 標本モーメント=理論モーメントを解く
      • ベイズ推定(事後平均・MAP)— ベイズ法ページ

    • 漸近理論
      • 最尤推定量の漸近正規性・漸近有効性
    • 区間推定

評価基準定義直感
不偏性E[θ̂]=θ狙いの中心がずれていない
一致性n→∞ で θ̂→θデータを増やせば当たる(大数の法則が根拠)
有効性不偏の中で分散最小(CR下限達成)同じ狙いなら散らばりが小さい方が良い
十分性θ̂ を知れば元データに θ の追加情報なし圧縮しても情報を失わない要約
十分統計量はなぜ学ぶのか: 「データ全部を持ち歩かなくても、この要約さえあれば母数の情報は一切失われない」という究極の圧縮の保証。例えばコイン投げ n回の結果は、成功回数 Σxᵢ さえあれば個々の順序は不要(Σxᵢ が p の十分統計量)。良い推定量は十分統計量の関数として作れる(ラオ・ブラックウェルの考え方)ため、「何を残せばよいか」の理論的な道しるべになる。判定には分解定理 f(x;θ)=g(T(x);θ)h(x) を使う。

最尤法 — 「このデータが最も出やすい母数」を選ぶ

$$L(\theta)=\prod_{i=1}^{n} f(x_i;\theta)\;\text{を最大化。実際は対数尤度}\;\; \ell(\theta)=\sum_{i=1}^{n}\ln f(x_i;\theta)\;\text{を微分して0}$$

手順は機械的:①尤度を書く → ②対数を取る(積が和になり微分が楽) → ③θで微分して0 → ④解く。代表例は暗記推奨:正規の μ̂=x̄・σ̂²=(1/n)Σ(xᵢ−x̄)²、ポアソンの λ̂=x̄、二項の p̂=x/n、指数の λ̂=1/x̄。

最尤推定量の便利な性質:不変性 — θ̂ がMLEなら g(θ̂) も g(θ) のMLE(例: σ̂²のMLEから σ̂=√σ̂² が即座に出る)。
対数尤度カーブ・ラボ(ポアソン分布)— 「データが増えると尤度が尖る」を見る

試してほしい操作:n を 5 → 100 へ → カーブが鋭く尖り、赤線(95%基準)と交わる幅=尤度比信頼区間がぐっと狭まる(n を4倍にすると幅は約半分 — √n の法則)。x̄ を動かす → 山の頂上(MLE λ̂=x̄)が追従。n が小さいとき → 区間が左右非対称(尤度カーブ自体が歪んでいる)なのに、n を上げると対称に近づく — Wald区間(±1.96SE)が「大標本でだけ正当化される近似」であることが見える。

🔧 実装メモ(このラボが実際に計算していること)
  • モデル: ポアソン分布のiid標本。対数尤度は ℓ(λ)=n(x̄ ln λ − λ)+C(定数Cは形に無関係なので省略)。データは x̄ と n だけで尤度が決まる(Σxᵢ が十分統計量である、の実演)
  • 描画: ℓ(λ)−ℓ(λ̂) を λ∈[0.2, 10](600点グリッド)で計算。y軸は[−8, 0.3]固定、はみ出しクリップ
  • 尤度比95%区間: ℓ(λ̂)−ℓ(λ) ≤ 1.92(χ²(1)の95%点 3.84 の半分)を満たす範囲をグリッド走査で近似(精度は格子幅≈0.016)
  • Wald区間 = λ̂ ± 1.96√(λ̂/n)(観測情報量 n/λ̂ の逆数が漸近分散)。比較のためreadoutに併記
  • 乱数は使っていない(x̄を直接動かす決定的な可視化)。正規・二項など他分布の尤度カーブは未実装

導出を1回だけ手で追う — 正規分布のMLEとバイアス

最尤法の計算は型が決まっているので、代表例を1回だけ丁寧に追えば全部に応用が利く。正規分布 N(μ, σ²) の対数尤度は:

$$\ell(\mu,\sigma^2) = -\frac{n}{2}\ln(2\pi\sigma^2) - \frac{\sum_i (x_i-\mu)^2}{2\sigma^2}$$

μで微分して0: Σ(xᵢ−μ)=0 → μ̂ = x̄。σ²で微分して0(σ²を1つの変数と見るのがコツ):

$$-\frac{n}{2\sigma^2} + \frac{\sum_i (x_i-\bar{x})^2}{2\sigma^4} = 0 \quad\Longrightarrow\quad \hat\sigma^2_{\mathrm{MLE}} = \frac{1}{n}\sum_i (x_i-\bar{x})^2$$

そしてこのMLEは不偏でないことも1行で示せる。恒等式 Σ(xᵢ−μ)² = Σ(xᵢ−x̄)² + n(x̄−μ)² の期待値を取ると、左辺は nσ²、右辺第2項は n·V[x̄]=σ² なので:

$$E\Big[\sum_i (x_i-\bar{x})^2\Big] = n\sigma^2 - \sigma^2 = (n-1)\,\sigma^2 \quad\Longrightarrow\quad E[\hat\sigma^2_{\mathrm{MLE}}] = \frac{n-1}{n}\,\sigma^2 < \sigma^2$$

x̄ に「σ²の1個分」の情報を使ってしまった分だけ系統的に小さい — 記述統計の基礎で直感として述べた「n−1で割る理由」の、これが厳密な姿。n−1 で割り直したものが不偏分散になる。

フィッシャー情報量とクラメール・ラオの不等式 — 精度の理論限界

$$I(\theta)=E\!\left[\left(\frac{\partial \ell_1}{\partial\theta}\right)^{\!2}\right]=-E\!\left[\frac{\partial^2 \ell_1}{\partial\theta^2}\right], \qquad V[\hat\theta] \;\ge\; \frac{1}{n\,I(\theta)} \;\;(\text{CR下限})$$

フィッシャー情報量は「データ1個が母数について持つ情報の量」。CR不等式はどんな不偏推定量もこれ以上は精度を上げられないという理論限界で、下限を達成する推定量が「有効推定量」。例: ポアソンの I(λ)=1/λ、CR下限=λ/n — x̄ の分散はちょうど λ/n なので x̄ は有効。

最尤推定量の漸近正規性 — 大標本での万能保証

$$\sqrt{n}\,\big(\hat\theta_{\mathrm{MLE}}-\theta\big) \xrightarrow{\;d\;} N\!\left(0,\;\frac{1}{I(\theta)}\right) \qquad (n\to\infty)$$

正則条件の下で、MLEは一致性をもち、漸近的に正規分布し、しかも漸近分散がCR下限に一致する(漸近有効)。「MLEを使っておけば大標本では最良」の理論的根拠であり、この漸近分散から近似信頼区間 θ̂ ± z·1/√(nI(θ̂)) が作れる。デルタ法と組み合わせれば g(θ̂) の区間も出る。

頻出の引っかけ:①正規分布の分散のMLE (1/n)Σ(xᵢ−x̄)² は不偏でない(不偏分散は n−1 で割る。MLE≠不偏の代表例)。ただし一致性はある。②「不偏で一致」「不偏でないが一致」「不偏だが一致でない」の組み合わせを判定させる問題。③CR下限は「不偏」推定量への限界 — バイアスを許せば下回れる(リッジ回帰などの発想)。

区間推定 — 枢軸量で「幅」を付ける

$$\text{枢軸量の例: } \frac{\bar{x}-\mu}{s/\sqrt{n}} \;\sim\; t(n-1) \quad (\text{分布が母数によらない})$$

枢軸量の分布の95%範囲を母数について解き直すと信頼区間になる。基本形は推定値 ± 分位点 × 標準誤差。母平均(σ未知)は t 分布、母分散は (n−1)s²/σ² ~ χ²(n−1)、母比率は正規近似 p̂±z√(p̂(1−p̂)/n)。「95%」は手順の当たり率(θは定数)であり、ベイズの信用区間との解釈の違いが問われる。

ポアソン分布 Po(λ) からの標本 x₁,…,xₙ について、λ の最尤推定量を導出せよ。
解答を見る

ℓ(λ) = Σ(−λ + xᵢ ln λ − ln xᵢ!) = −nλ + (Σxᵢ)ln λ + C
dℓ/dλ = −n + Σxᵢ/λ = 0 → λ̂ = x̄
(二階微分 −Σxᵢ/λ² < 0 で最大値であることも確認)

正規母集団からの n=16 の標本で x̄=25.0, 不偏分散 s²=16 のとき、母平均 μ の95%信頼区間を求めよ(t₀.₀₂₅(15)=2.131)。
解答を見る

x̄ ± t·s/√n = 25.0 ± 2.131×4/4 = 25.0 ± 2.131 → [22.87, 27.13]

指数分布 f(x;λ)=λe^{−λx} からの標本 x₁,…,xₙ について、(a) λ のMLEを導出せよ。(b) フィッシャー情報量 I(λ)=1/λ² を用いて λ̂ の漸近分散を答えよ。
解答を見る

(a) ℓ(λ)=n ln λ − λΣxᵢ。微分して0: n/λ − Σxᵢ = 0 → λ̂ = 1/x̄
(b) 漸近分散 = 1/(nI(λ)) = λ²/n。つまり √n(λ̂−λ) → N(0, λ²)。区間は λ̂ ± 1.96 λ̂/√n で近似できる。

上のラボで x̄=3, n=20 のときのWald 95%信頼区間を手計算し、ラボの表示と照合せよ。
解答を見る

λ̂=3, SE=√(λ̂/n)=√0.15≈0.387。区間 = 3 ± 1.96×0.387 = [2.24, 3.76]。
ラボの尤度比区間はこれよりわずかに右に寄る(尤度カーブの非対称性)。n=80 にすると両者はほぼ一致する。
