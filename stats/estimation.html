全体マップ — この分野の体系

推定の問いは2つ:「どう作るか(推定量の構成法)」と「どれが良いか(推定量の評価基準)」。準1級最頻出の最尤法は構成法の王様で、その良さ(漸近有効性)は評価理論(クラメール・ラオ)で保証される — 両者が噛み合う構造を掴むこと。

最尤法 — 「このデータが最も出やすい母数」を選ぶ

L(θ)=∏f(xᵢ;θ) を最大化。実際は対数尤度 ℓ(θ)=Σln f(xᵢ;θ) を微分して0(尤度方程式)

手順は機械的:①尤度を書く → ②対数を取る(積が和になり微分が楽) → ③θで微分して0 → ④解く。代表例は暗記推奨:正規の μ̂=x̄・σ̂²=(1/n)Σ(xᵢ−x̄)²、ポアソンの λ̂=x̄、二項の p̂=x/n、指数の λ̂=1/x̄。

最尤推定量の便利な性質:不変性 — θ̂ がMLEなら g(θ̂) も g(θ) のMLE(例: σ̂²のMLEから σ̂=√σ̂² が即座に出る)。

フィッシャー情報量とクラメール・ラオの不等式 — 精度の理論限界

I(θ)=E[(∂ℓ/∂θ)²]=−E[∂²ℓ/∂θ²], 不偏推定量の分散 ≥ 1/(nI(θ))(CR下限)

フィッシャー情報量は「データ1個が母数について持つ情報の量」。CR不等式はどんな不偏推定量もこれ以上は精度を上げられないという理論限界で、下限を達成する推定量が「有効推定量」。例: ポアソンの I(λ)=1/λ、CR下限=λ/n — x̄ の分散はちょうど λ/n なので x̄ は有効。

最尤推定量の漸近正規性 — 大標本での万能保証

√n(θ̂_MLE − θ) → N(0, 1/I(θ)) (n→∞)

正則条件の下で、MLEは一致性をもち、漸近的に正規分布し、しかも漸近分散がCR下限に一致する(漸近有効)。「MLEを使っておけば大標本では最良」の理論的根拠であり、この漸近分散から近似信頼区間 θ̂ ± z·1/√(nI(θ̂)) が作れる。デルタ法と組み合わせれば g(θ̂) の区間も出る。

頻出の引っかけ:①正規分布の分散のMLE (1/n)Σ(xᵢ−x̄)² は不偏でない(不偏分散は n−1 で割る。MLE≠不偏の代表例)。ただし一致性はある。②「不偏で一致」「不偏でないが一致」「不偏だが一致でない」の組み合わせを判定させる問題。③CR下限は「不偏」推定量への限界 — バイアスを許せば下回れる(リッジ回帰などの発想)。

区間推定 — 枢軸量で「幅」を付ける

枢軸量: 分布が母数によらない量。例: (x̄−μ)/(s/√n) ~ t(n−1)

枢軸量の分布の95%範囲を母数について解き直すと信頼区間になる。基本形は推定値 ± 分位点 × 標準誤差。母平均(σ未知)は t 分布、母分散は (n−1)s²/σ² ~ χ²(n−1)、母比率は正規近似 p̂±z√(p̂(1−p̂)/n)。「95%」は手順の当たり率(θは定数)であり、ベイズの信用区間との解釈の違いが問われる。

ポアソン分布 Po(λ) からの標本 x₁,…,xₙ について、λ の最尤推定量を導出せよ。
解答を見る

ℓ(λ) = Σ(−λ + xᵢ ln λ − ln xᵢ!) = −nλ + (Σxᵢ)ln λ + C
dℓ/dλ = −n + Σxᵢ/λ = 0 → λ̂ = x̄
(二階微分 −Σxᵢ/λ² < 0 で最大値であることも確認)

正規母集団からの n=16 の標本で x̄=25.0, 不偏分散 s²=16 のとき、母平均 μ の95%信頼区間を求めよ(t₀.₀₂₅(15)=2.131)。
解答を見る

x̄ ± t·s/√n = 25.0 ± 2.131×4/4 = 25.0 ± 2.131 → [22.87, 27.13]
