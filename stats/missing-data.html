欠測値の統計処理:メカニズムの分類と対処

タグ: 欠測 / MCAR / MAR / MNAR / リストワイズ削除 / 単一代入 / 多重代入 / EMアルゴリズム

※導入版(フェーズ1): 全体像と主要概念・例題を押さえるページ。

全体マップ — この分野の体系

欠測処理の要点はただ1つ:「なぜ欠けたのか(メカニズム)で、許される対処が決まる」。手法の暗記より先に、MCAR/MAR/MNARの3分類を判定できるようになることがすべての出発点。

  • 欠測値の統計処理
    • 欠測メカニズムの分類 — まずこれを判定する
      • MCAR — 完全にランダムな欠測
      • MAR — 観測データに依存する欠測
      • MNAR — 欠けた値そのものに依存する欠測(最難)
    • 素朴な対処とその問題
      • リストワイズ削除・平均値代入の落とし穴
    • 適切な対処
      • 多重代入(MI)の考え方
      • EMアルゴリズム・最尤法ベースの触り
      • 逆確率重み付け(IPW)— 未収載(因果推論の傾向スコアと同系)
メカニズム欠測確率が依存するもの例(所得の欠測)リストワイズ削除MAR前提の手法(MI等)
MCAR何にも依存しない回答用紙をランダムに紛失不偏(nが減るだけ)OK
MAR観測されている変数高齢者ほど無回答(年齢は観測済み)バイアスOK(年齢で調整可能)
MNAR欠測した値そのもの高所得者ほど所得を無回答バイアス不十分(感度分析が必要)

素朴な対処がダメな理由

リストワイズ削除(欠測のある行を捨てる)はMCARでない限り偏り、しかもnが減って検出力も落ちる。平均値代入はさらに悪質で、同じ値を埋めるため分散が過小評価され、相関も歪み、「自信過剰な(狭すぎる)信頼区間」を生む — 代入した値を実測値のように扱うことが不確実性の無視にあたる。

多重代入 — 「わからなさ」ごと持ち運ぶ

①観測データに基づく予測分布から欠測値を乱数でm回(例: m=20)代入し、m個の完成データを作る → ②それぞれ普通に解析 → ③結果を統合(ルービンの規則: 推定値は平均、分散は「代入内分散+代入間分散」)。代入間のばらつきが「欠測による不確実性」を表現するのが単一代入との決定的な違い。MARの下で妥当。

EMアルゴリズムは代入とは別ルートで、「欠測値の条件付き期待値で埋める(Eステップ)→母数を最尤推定(Mステップ)」を収束まで交互に繰り返し、観測データの尤度を直接最大化する。MARの下で一致推定量を与える。k-meansの割当↔重心更新と同じ「交互最適化」の構造。
MNARは観測データだけからはMARと区別できない(欠けた値は見えないから)。だから実務では「MARを仮定して解析+仮定が崩れた場合の感度分析」が標準作法。この限界の認識自体が出題される。
次の欠測はMCAR/MAR/MNARのどれか:(a)健康調査で、体重が重い人ほど体重の回答を拒否 (b)若い回答者ほど収入項目が無回答(年齢は全員回答済み) (c)機器の故障でランダムな日の測定値が消失
解答を見る

(a) MNAR(欠けた値=体重そのものに依存) (b) MAR(観測済みの年齢に依存) (c) MCAR(データと無関係)。
(b)は年齢を使った多重代入等で対処可能、(a)は感度分析が必要。

平均値代入(欠測をすべて標本平均で埋める)を行うと、分散と相関係数はどう歪むか。
解答を見る

分散: 平均ちょうどの値が増えるので過小評価。相関: 代入された点は回帰的な関係に寄与しない水平な点列になるため0に向かって弱まる(希薄化)。結果として標準誤差も過小 → 有意になりやすい危険な方向に歪む。
