条件付き確率と独立 — 「情報が入ると確率が変わる」

P(A|B) = P(A∩B)/P(B), 独立 ⇔ P(A∩B)=P(A)P(B) ⇔ P(A|B)=P(A)

全確率の公式 P(A)=ΣP(A|Bᵢ)P(Bᵢ)(排反な原因Bᵢで場合分け)と、その逆向きのベイズの定理はセットで使う。「排反」(同時に起きない)と「独立」(情報にならない)の混同は定番の引っかけ — 排反な2事象は(確率が正なら)必ず従属。

期待値・分散の演算規則 — 全分野で使う筋力

E[aX+bY]=aE[X]+bE[Y](常に成立) / V[aX+b]=a²V[X] / V[X±Y]=V[X]+V[Y]±2Cov(X,Y)

期待値の線形性は独立でなくても成立する(これが強力)。分散は独立(無相関)のときだけ足し算になる。V[X]=E[X²]−(E[X])² は計算の主力。共分散は Cov(X,Y)=E[XY]−E[X]E[Y]。

「独立 ⇒ 無相関」は成り立つが逆は不成立(例: X~N(0,1), Y=X² は無相関だが従属)。また V[X−Y] の符号ミス(−2Covを忘れて引き算)が最頻出の計算事故。

母関数 — 分布を関数1つに圧縮する

MGF: M(t)=E[e^{tX}], E[Xᵏ]=M⁽ᵏ⁾(0)。独立和: M_{X+Y}(t)=M_X(t)M_Y(t)

べき級数展開 e^{tX}=Σ(tX)ᵏ/k! の係数にモーメントが並ぶので、微分して0を代入するとモーメントが取り出せる。独立な和のMGFが積になる性質は、再生性(二項+二項、正規+正規、ポアソン+ポアソン)の統一的な証明手段。準1級では「MGFからE[X], V[X]を求めよ」「MGFの形から分布を同定せよ」が頻出。

代表的なMGF: 正規 N(μ,σ²) → exp(μt+σ²t²/2) / ポアソン Po(λ) → exp(λ(eᵗ−1)) / 二項 B(n,p) → (1−p+peᵗ)ⁿ / 指数 Exp(λ) → λ/(λ−t) (t<λ)。この4つは形を見て即答できるように。

変数変換 — 分布の「座標変換」

Y=g(X)(単調)のとき f_Y(y) = f_X(g⁻¹(y)) · |d g⁻¹(y)/dy|

「逆関数を代入して、伸び縮み率(微分の絶対値)を掛ける」。2変数では伸び縮み率がヤコビアン(偏微分の行列式)になる。単調でない変換(Y=X²など)は区間を分けて足す。累積分布関数を経由する方法(P(Y≤y)を直接計算して微分)はどんな場合でも使える万能ルート。

極限定理 — nが大きいときの普遍法則

大数の法則: X̄ₙ → μ / 中心極限定理: √n(X̄ₙ−μ)/σ → N(0,1)

大数の法則は「標本平均が真の値に収束する」(一致性の根拠)、中心極限定理は「そのゆらぎ方が正規分布になる」(検定・区間推定の根拠)。役割の違いを言い分けられることが大切。さらに準1級では、推定量の関数の漸近分布を出すデルタ法まで問われる:

デルタ法: √n(g(X̄ₙ)−g(μ)) → N(0, [g′(μ)]²σ²)

「g で変換すると、分散は微分係数の2乗倍になる」— テイラー展開の1次近似 g(X̄)≈g(μ)+g′(μ)(X̄−μ) がそのまま結論になっている。

指数分布 Exp(λ) のMGFは M(t)=λ/(λ−t) (t<λ)。これを使って E[X] と V[X] を求めよ。
解答を見る

M′(t)=λ/(λ−t)² より E[X]=M′(0)=1/λ。
M″(t)=2λ/(λ−t)³ より E[X²]=M″(0)=2/λ²。
V[X]=2/λ²−(1/λ)²=1/λ²。

X̄ₙ が平均 μ=2、分散 σ²=4 の分布からの標本平均のとき、g(x)=x² としたときの √n(X̄ₙ²−4) の漸近分布をデルタ法で求めよ。
解答を見る

g′(x)=2x, g′(μ)=4。デルタ法より √n(X̄ₙ²−μ²) → N(0, g′(μ)²σ²) = N(0, 16×4=64)。
