条件付き確率と独立 — 「情報が入ると確率が変わる」

$$P(A\mid B)=\frac{P(A\cap B)}{P(B)}, \qquad \text{独立} \iff P(A\cap B)=P(A)\,P(B) \iff P(A\mid B)=P(A)$$

全確率の公式 P(A)=ΣP(A|Bᵢ)P(Bᵢ)(排反な原因Bᵢで場合分け)と、その逆向きのベイズの定理はセットで使う。「排反」(同時に起きない)と「独立」(情報にならない)の混同は定番の引っかけ — 排反な2事象は(確率が正なら)必ず従属。

期待値・分散の演算規則 — 全分野で使う筋力

$$E[aX+bY]=aE[X]+bE[Y] \quad(\text{常に成立}), \qquad V[aX+b]=a^2V[X]$$ $$V[X\pm Y]=V[X]+V[Y]\pm 2\,\mathrm{Cov}(X,Y)$$

期待値の線形性は独立でなくても成立する(これが強力)。分散は独立(無相関)のときだけ足し算になる。V[X]=E[X²]−(E[X])² は計算の主力。共分散は Cov(X,Y)=E[XY]−E[X]E[Y]。

「独立 ⇒ 無相関」は成り立つが逆は不成立(例: X~N(0,1), Y=X² は無相関だが従属)。また V[X−Y] の符号ミス(−2Covを忘れて引き算)が最頻出の計算事故。

母関数 — 分布を関数1つに圧縮する

なぜ学ぶのか: 期待値や分散を定義どおり計算すると、分布ごとに面倒な和や積分をやり直すことになる。母関数はその手間を一掃する道具で、①モーメント(E[X], E[X²], …)が「微分して0を代入」だけで機械的に出る、②独立な和の分布が関数の掛け算で求まる(再生性の証明が一行になる)、③形を見れば分布を同定できる(母関数は分布を一意に決める)、④中心極限定理などの極限定理の証明の主役になる。準1級では特に①(積率母関数からのモーメント計算)と②が頻出。

$$M(t)=E\big[e^{tX}\big], \qquad E[X^k]=M^{(k)}(0), \qquad M_{X+Y}(t)=M_X(t)\,M_Y(t)\;(\text{独立})$$

べき級数展開 e^{tX}=Σ(tX)ᵏ/k! の係数にモーメントが並ぶので、微分して0を代入するとモーメントが取り出せる。独立な和のMGFが積になる性質は、再生性(二項+二項、正規+正規、ポアソン+ポアソン)の統一的な証明手段。準1級では「MGFからE[X], V[X]を求めよ」「MGFの形から分布を同定せよ」が頻出。

代表的なMGF: 正規 N(μ,σ²) → exp(μt+σ²t²/2) / ポアソン Po(λ) → exp(λ(eᵗ−1)) / 二項 B(n,p) → (1−p+peᵗ)ⁿ / 指数 Exp(λ) → λ/(λ−t) (t<λ)。この4つは形を見て即答できるように。

変数変換 — 分布の「座標変換」

なぜ学ぶのか: 実際に興味がある量は、観測そのものより「観測の関数」であることが多い(半径の分布から面積の分布、対数変換した収益、Z=(X−μ)/σ の標準化…)。X の分布が分かっているとき Y=g(X) の分布を作る一般手順が変数変換で、標準正規からχ²を作る(X²)など分布の家系図はすべてこの操作で繋がっている。

$$Y=g(X)\;(\text{単調})\;\text{のとき}\quad f_Y(y) = f_X\!\big(g^{-1}(y)\big)\,\left|\frac{d\,g^{-1}(y)}{dy}\right|$$

「逆関数を代入して、伸び縮み率(微分の絶対値)を掛ける」。2変数では伸び縮み率がヤコビアン(偏微分の行列式)になる。単調でない変換(Y=X²など)は区間を分けて足す。累積分布関数を経由する方法(P(Y≤y)を直接計算して微分)はどんな場合でも使える万能ルート。

極限定理 — nが大きいときの普遍法則

$$\text{大数の法則: } \bar{X}_n \xrightarrow{\;p\;} \mu \qquad \text{中心極限定理: } \frac{\sqrt{n}\,(\bar{X}_n-\mu)}{\sigma} \xrightarrow{\;d\;} N(0,1)$$

大数の法則は「標本平均が真の値に収束する」(一致性の根拠)、中心極限定理は「そのゆらぎ方が正規分布になる」(検定・区間推定の根拠)。役割の違いを言い分けられることが大切。

中心極限定理シミュレータ — どんな分布でも「平均」は正規に向かう

試してほしい操作:指数分布で n=1 → ヒストグラムは元の分布の形そのまま(強い右歪み)。n を 5, 30 と上げる → 歪みが消えて赤点線(標準正規)にぴったり重なっていく — これが中心極限定理の実演。二峰分布(正規から最も遠い形)でも n=30 でほぼ正規になることを確認。「元の分布が何であっても」の凄みが分かる。

🔧 実装メモ(このシミュレータが実際に計算していること)
  • 手順: 選択した分布から n 個抽出→標本平均→標準化 z=(x̄−μ)/(σ/√n)、を10,000回繰り返し、z のヒストグラム(幅0.2、範囲[−4,4])を描画
  • 元の分布と(μ, σ²): 指数(λ=1): (1, 1)、逆関数法 −ln(1−U) で生成 / 一様[0,1]: (0.5, 1/12) / 二峰: ±1.5 を等確率で選び N(±1.5, 0.5²)、(0, 2.5)
  • 赤点線: 標準正規 N(0,1) の密度(理論曲線)。ヒストグラムは密度スケール(度数/(回数×ビン幅))
  • 軸固定: x∈[−4,4]、y∈[0, 0.6]。正規乱数はBox–Muller法

さらに準1級ではデルタ法まで問われる。なぜ必要か: 実際に報告したい量は推定量そのものではなく、その関数であることが多い(比率の推定値からオッズ p/(1−p)、分散から標準偏差、係数から e^β…)。CLTが教えてくれるのは X̄ の分布だけなので、その関数 g(X̄) の分布(=誤差の大きさ)を出す変換規則が要る — それがデルタ法:

$$\text{デルタ法: } \sqrt{n}\,\big(g(\bar{X}_n)-g(\mu)\big) \xrightarrow{\;d\;} N\!\big(0,\,[g'(\mu)]^2\sigma^2\big)$$

「g で変換すると、分散は微分係数の2乗倍になる」— テイラー展開の1次近似 g(X̄)≈g(μ)+g′(μ)(X̄−μ) がそのまま結論になっている。

指数分布 Exp(λ) のMGFは M(t)=λ/(λ−t) (t<λ)。これを使って E[X] と V[X] を求めよ。
解答を見る

M′(t)=λ/(λ−t)² より E[X]=M′(0)=1/λ。
M″(t)=2λ/(λ−t)³ より E[X²]=M″(0)=2/λ²。
V[X]=2/λ²−(1/λ)²=1/λ²。

X̄ₙ が平均 μ=2、分散 σ²=4 の分布からの標本平均のとき、g(x)=x² としたときの √n(X̄ₙ²−4) の漸近分布をデルタ法で求めよ。
解答を見る

g′(x)=2x, g′(μ)=4。デルタ法より √n(X̄ₙ²−μ²) → N(0, g′(μ)²σ²) = N(0, 16×4=64)。
