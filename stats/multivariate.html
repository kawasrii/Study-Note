多変量解析:主成分・判別・クラスター

タグ: 主成分分析 PCA / 固有値 / 寄与率 / 判別分析 LDA / マハラノビス距離 / クラスター分析 / k-means

全体マップ — 多変量解析は「目的」で分類する

変数が多いデータを扱う手法群。分類の軸は①目的(縮約か・分類か・関係の記述か)と②正解ラベル(教師)の有無。この2つで見れば、名前が似ていて混ざりがちな判別分析(教師あり)とクラスター分析(教師なし)も混同しない。

  • 多変量解析(たくさんの変数をまとめて扱う)
    • 次元縮約 — 情報をなるべく保ったまま変数を減らす(教師なし)
      • 主成分分析(PCA) — 分散最大の方向に軸を取り直す(ラボあり)
      • 因子分析 — 背後の潜在因子を仮定 — 未収載
    • 分類
      • 教師あり(正解ラベルを使って境界を学ぶ)
        • 判別分析(LDA) — クラス間/クラス内分散比を最大化(解説+例題)
        • ロジスティック回帰による分類 — 未収載
      • 教師なし(ラベルなしで塊を見つける)
        • k-means法 — 重心との距離で分割を反復改善(ラボあり)
        • 階層的クラスタリング(デンドログラム)— 解説のみ
    • 関係の記述 — 正準相関・多次元尺度構成法(MDS)— 未収載
手法目的教師最大化/最小化するもの出力
主成分分析次元縮約なし射影後の分散を最大化主成分軸・寄与率・主成分得点
判別分析分類(境界を学ぶ)ありクラス間分散/クラス内分散の比を最大化判別関数・判別境界

主成分分析(PCA) — 分散が最大の方向に軸を取り直す

相関のある変数たちを、互いに無相関な新しい軸(主成分)に変換する。第1主成分はデータの分散(ばらつき)が最大になる方向、第2主成分はそれと直交する中で分散最大の方向…と取っていく。数学的には分散共分散行列(または相関行列)の固有値問題で、固有ベクトルが軸の向き、固有値 λ がその軸方向の分散になる。

第k主成分の寄与率 = λₖ / Σλ, 累積寄与率 = (λ₁+…+λₖ) / Σλ

「累積寄与率80%まで採用」のように使い、10変数を2〜3軸に圧縮して可視化・解釈する。変数の単位が違うときは標準化(相関行列ベース)が必須 — しないと分散の大きい変数が主成分を独占する。

PCAラボ — クリック/タップで点を追加(もう一度で削除)。矢印=主成分軸
点を3つ以上打つと主成分軸が現れます。

試してほしい操作:「サンプル(相関あり)」→ 長い矢印(第1主成分)が点群の伸びている方向を向く。矢印の長さは √固有値 に比例=その方向の標準偏差。「射影を表示」→ 各点が第1主成分軸上に潰される — これが「2次元→1次元への縮約」の実体で、失う情報は軸への垂線の長さ(第2主成分方向の分散)。「相関ほぼなし」→ 2つの固有値が近くなり、寄与率が50%ずつに — 縮約しても旨みがない状況。点を1つずつ足して軸がグリッと回る様子も見てほしい。

🔧 実装メモ(このラボが実際に計算していること)
  • 分散共分散行列(2×2)を標本から計算(不偏分散、n−1 で割る)。標準化はしていない(このラボは両軸とも同じ0–10の尺度のため生の共分散行列を使用。単位が違う実データでは相関行列ベースにすべき、という本文の注意はこのため)
  • 固有値・固有ベクトル: 2×2対称行列の閉じた式 λ = (a+c)/2 ± √(((a−c)/2)²+b²)。数値反復(QR法等)は不使用
  • 矢印: 重心から固有ベクトル方向に ±2√λ(=±2標準偏差)の長さで描画
  • 寄与率 = λ₁/(λ₁+λ₂)。readoutに固有値とともに表示
  • 射影: 各点 x を 重心 + v₁v₁ᵀ(x−重心) に写して空心円で表示(v₁は第1固有ベクトル)
  • 3変数以上・標準化の切替・主成分得点のプロット(バイプロット)は未実装

判別分析(LDA) — ラベルが最も分かれて見える方向

2クラスのラベル付きデータで、クラス間のずれ(群間分散)がクラス内のばらつき(群内分散)に比べて最大になる方向 w に射影して分類する:

w ∝ S_pooled⁻¹ (μ₁ − μ₂), 新しい点は射影値が判別点(中点)のどちら側かで分類

PCAと見比えると対比が鮮明:PCAはラベル無視で「ばらつき最大」の方向、LDAはラベルを使って「分離最大」の方向。2つの方向は一般に一致しない — 分散が大きい方向とクラスが分かれる方向は別物だから。等分散の正規母集団を仮定すると、LDAはマハラノビス距離(共分散を考慮した距離)が近い方のクラスに割り当てる操作と等価になる。

準1級では「判別関数の係数の意味」「マハラノビス距離 D² = (x−μ)ᵀΣ⁻¹(x−μ)(ばらつきの大きい方向のずれは割り引く)」「誤判別率」が問われる。判別分析は教師あり・クラスターは教師なしという区別は選択肢問題の定番。
2つの主成分の固有値が λ₁=3.2, λ₂=0.8 のとき、第1主成分の寄与率は?また元の2変数の分散の合計は?
解答を見る

寄与率 = 3.2/(3.2+0.8) = 80%
固有値の和は元の分散の和(トレース)に等しいので 3.2+0.8 = 4.0。「固有値の和=全分散」はPCAの核心的性質。

クラスター分析 — ラベルなしで塊を見つける

代表格の k-means法 は「①k個の重心を仮置き → ②各点を最寄りの重心に割当 → ③各クラスタの平均に重心を移動 → ②③を収束まで反復」というシンプルな交互最適化で、クラスタ内二乗和(WCSS)を単調に減らす。ただし局所解に落ちる(初期値で結果が変わる)、kは自分で決める(エルボー法などで選ぶ)、球状の塊しか見つけにくい、という3つの弱点も頻出論点。

もう1つの系統が階層的クラスタリング:近いもの同士を順に併合してデンドログラム(樹形図)を作る。結合基準(最短距離法・最長距離法・ウォード法)で結果が変わる。kを事前に決めなくてよいのが利点。

k-meansラボ — 点を打って「実行」。初期値で結果が変わることも体験する
点を打って「実行」を押してください。

試してほしい操作:「サンプル(3つの塊)」→ k=3 で実行 → 塊どおりに色分けされ、×印(重心)が各塊の中心に立つ。「実行」を何度も押す → 初期値が毎回ランダムなので、たまに変な分かれ方(局所解)になる — WCSSの値が実行ごとに違うことで確認できる。k=2 や k=4 にして実行 → kを間違えると塊が併合・分裂する。kはアルゴリズムが教えてくれない、という弱点の体感。

🔧 実装メモ(このラボが実際に計算していること)
  • アルゴリズム: Lloyd法(標準のk-means)。割当→重心更新を割当が変化しなくなるまで反復(上限50回)
  • 初期値: データ点から重複なしの一様ランダム抽出(k-means++ は未実装 — 局所解に落ちやすいのはこのため。それ自体を教材にしている)
  • 距離: ユークリッド距離の2乗。空クラスタが生じた場合はランダムな点を重心に再割当
  • WCSS = Σ 各点から所属重心への距離²。readoutに反復回数とともに表示
  • エルボー法のためのk別WCSSプロット・k-means++・階層的クラスタリング(デンドログラム)は未実装
k-means法について正しいものを選べ:①WCSSは反復のたびに増加する ②kは自動的に決まる ③初期値によって結果が変わることがある ④教師ありの手法である
解答を見る

③が正しい。①WCSSは単調に「減少」する(割当も重心更新もWCSSを下げる操作)。②kは事前に指定が必要(エルボー法などで選ぶ)。④ラベルを使わない「教師なし」学習。上のラボで①③は実際に確認できる。
