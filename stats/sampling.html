標本調査法:抽出デザインと推定精度

タグ: 単純無作為抽出 / 有限母集団修正 / 層化抽出 / 比例配分 / ネイマン配分 / 集落抽出 / 多段抽出 / 系統抽出

フェーズ2対応済み: 層化抽出vs単純無作為抽出の精度比較シミュレータつき。

前提知識: 標本平均の分散 V(x̄)=σ²/n の由来は確率と確率変数(分散の演算規則)、層化の理屈の親玉「全分散の公式」は多次元確率分布にあります。

全体マップ — この分野の体系

「母集団の誰を・どう選ぶか」の設計論。これまでの理論は「無限母集団からのiid標本」を仮定していたが、実際の調査は有限母集団から非復元で抽出する — そこで生まれる修正と、精度・コストを両立させる工夫(層化・集落・多段)が主題。

  • 標本調査法
    • 確率抽出(無作為性が保証される)
      • 単純無作為抽出(SRS)と有限母集団修正 — 基準となる方式
      • 層化抽出 — 均質な層に分けてから抽出(精度向上)
      • 集落(クラスター)抽出 — 塊ごと抽出(コスト減、精度は通常低下)
      • 多段抽出・系統抽出
    • 非確率抽出
      • 有意抽出・割当法 — 誤差が評価できない(注意点のみ)
    • 推定の工夫
      • 比推定・回帰推定 — 補助情報の利用 — 未収載

単純無作為全員に等確率基準全員の名簿が必要

有限母集団修正(FPC) — 「たくさん抜けば誤差は減る」の定量化

$$V(\bar{x}) = \left(1-\frac{n}{N}\right)\frac{S^2}{n} \qquad (N:\ \text{母集団サイズ},\; n:\ \text{標本サイズ},\ \text{非復元})$$

無限母集団の S²/n に (1−n/N) が掛かる。n/N(抽出率)が大きいほど分散が縮み、全数調査(n=N)で誤差ゼロ。逆に抽出率が数%以下なら修正はほぼ1で無視してよい — 「母集団が1億でも1万でも、n=1000の精度はほぼ同じ」という直感に反する重要事実。

層化抽出 — 分散を「層間」ぶんだけタダで消す

$$\bar{x}_{st} = \sum_h \frac{N_h}{N}\,\bar{x}_h, \qquad V(\bar{x}_{st}) = \sum_h \left(\frac{N_h}{N}\right)^{\!2} V(\bar{x}_h)$$

全分散の公式で見た「全分散=層内+層間」のうち、層化は層間分散を推定誤差から消す。だから層内が均質(層間の差が大きい)ほど得をする。標本の層への配り方が設計の腕の見せ所:

$$\text{比例配分: } n_h \propto N_h \qquad \text{ネイマン(最適)配分: } n_h \propto N_h S_h$$

ネイマン配分は「大きい層・ばらつきの大きい層に多めに配る」— 分散最小の理論解。層内標準偏差 S_h が全部等しければ比例配分と一致する。

集落抽出が層化と逆に精度を落とすのは、同じ集落の個体は似ている(級内相関が正)ため、実質的な情報量が見かけの n より少ないから。「層化は層内を均質に、集落は集落間を均質にすると得」という対比が頻出。
N=2000 の母集団から n=400 を非復元で単純無作為抽出した。S²=100 のとき V(x̄) を求め、FPCを無視した場合と比較せよ。
解答を見る

V(x̄) = (1−400/2000)×100/400 = 0.8×0.25 = 0.2
無視すると 100/400 = 0.25。FPCで2割減 — 抽出率20%は無視できない水準。

2層からなる母集団: 層1(N₁=600, S₁=4)、層2(N₂=400, S₂=16)。総標本 n=100 をネイマン配分せよ。
解答を見る

N₁S₁=2400, N₂S₂=6400。n₁=100×2400/8800≈27, n₂=100×6400/8800≈73。
比例配分なら 60:40 だが、層2はばらつきが4倍大きいので大幅に増員される。
