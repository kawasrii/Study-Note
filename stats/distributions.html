確率分布ラボ

タグ: 正規分布 / 二項分布 / ポアソン分布 / t分布 / カイ二乗分布 / F分布 / 中心極限定理

準1級では「分布の定義を暗記している」だけでは足りず、パラメータを動かすと形がどう変わるか、分布同士がどうつながっているか(二項→ポアソン→正規、正規→χ²→t・F)を問われます。

前提知識: 各分布に出てくる E[X](期待値)・V[X](分散)の定義は確率変数と期待値・分散の定義を参照。

全体マップ — 6つの分布の位置づけ

分布は「何を数えている/測っているか(生成メカニズム)」で覚えると混ざらない。さらに、連続4分布は正規分布を親とする家系図(正規→二乗和がχ²→比を取るとt・F)になっており、これが推測統計の道具立てそのもの。まず階層で俯瞰する:

  • 確率分布(確率のかたち)
    • 離散型 — 数えられる値(0,1,2,…)の確率
      • 2値試行の繰り返し
        • ベルヌーイ(1回)— 未収載
        • 二項 B(n,p) — n回中の成功数
        • 幾何・負の二項(初成功までの回数など)— 未収載
      • 計数(カウント)
        • ポアソン Po(λ) — 稀な事象の回数。二項の極限
    • 連続型 — 実数値の確率密度
      • 観測データそのものの分布
        • 正規 N(μ,σ²) — 多数の要因の和(中心極限定理)。体系の中心
        • 一様・指数・ガンマ 等 — 未収載
      • 標本分布 — 統計量が従う分布。検定・推定の道具(正規の子どもたち)
        • χ²(k) — 標準正規の二乗和 → 分散の検定へ
        • t(ν) — 正規 ÷ √(χ²/ν) → 平均の検定へ
        • F(d₁,d₂) — χ² の比 → 分散比較・ANOVAへ
分布型生成メカニズムパラメータE[X] / V[X]他分布とのつながり
二項 B(n,p)離散n回中の成功数n, pnp / np(1−p)n大で正規へ、p小で ポアソンへ
ポアソン Po(λ)離散単位時間の稀な事象数λλ / λ二項の極限。λ大で正規へ
正規 N(μ,σ²)連続多数の要因の和(CLT)μ, σμ / σ²すべての親。再生性あり
χ²(k)連続標準正規の二乗和kk / 2k分散の推定・適合度検定の基盤
t(ν)連続正規 ÷ √(χ²/ν)ν0 / ν(ν−2)⁻¹·ν ※ν>2ν→∞で正規へ。t²=F(1,ν)
F(d₁,d₂)連続χ²比 (W₁/d₁)/(W₂/d₂)d₁, d₂d₂/(d₂−2) / — ※d₂>2分散比較・ANOVAの基盤

まず下のラボで形を体感してから、この表に戻って「つながり」を確認する、を往復するのが効率的です。

分布シミュレータ — 軸は固定スケール。パラメータを動かすと形の変化がそのまま見える

試してほしい操作:正規分布で μ を動かす → 山が横に平行移動する。σ を上げる → 低く・広くなる(面積は常に1)。二項分布で n を上げる → 山が右へ動きながら広がり、釣鐘型に近づく。t分布で ν を上げる → 裾が軽くなり正規分布に近づく。緑の点線は期待値 E[X] の位置。

🔧 実装メモ(このラボが実際に計算していること)
  • 確率(密度)関数は対数領域で計算して最後に exp(例: 二項係数は ln n! の差で計算)。桁あふれ・桁落ちの防止のため
  • ln Γ(x) は Lanczos 近似(g=7, 8項)。ln n! = ln Γ(n+1) として使用
  • 正規分布の cdf Φ は Abramowitz–Stegun 7.1.26 の有理近似(絶対誤差 < 7.5×10⁻⁸)による erf 近似
  • 軸の固定値: 正規 x∈[−10,10]・y上限0.85 / t x∈[−6,6]・0.45 / χ² x∈[0,50]・0.5 / F x∈[0,5]・1.6 / 二項 x∈[0,100] / ポアソン x∈[0,45](離散2つはy自動)。「軸を自動調整」チェックで従来型の追従軸にも切替可
  • 連続分布は301点グリッドで評価。離散分布の棒幅は格子間隔の70%
  • 「正規近似」の重ね描きは、選択中の分布と同じ期待値・分散をもつ正規pdf(赤点線)。連続修正は適用していない

正規分布 N(μ, σ²)

$$f(x)=\frac{1}{\sqrt{2\pi\sigma^2}}\exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right), \qquad E[X]=\mu,\;\; V[X]=\sigma^2$$

統計学の中心にある分布。μ が位置、σ が広がりを決める。μ±σ に約68%、μ±2σ に約95%、μ±3σ に約99.7% が入る(68-95-99.7則)。標準化 Z=(X−μ)/σ で N(0,1) に変換して確率計算するのが基本動作。

再生性をもつ:独立な X₁~N(μ₁,σ₁²), X₂~N(μ₂,σ₂²) に対して X₁+X₂ ~ N(μ₁+μ₂, σ₁²+σ₂²)。分散は引き算でも足される(X₁−X₂ の分散は σ₁²+σ₂²)点が頻出の引っかけ。

二項分布 B(n, p)

$$P(X=k)=\binom{n}{k}p^k(1-p)^{n-k}, \qquad E[X]=np,\;\; V[X]=np(1-p)$$

成功確率 p の試行を独立に n 回行った成功回数。上のラボで n を大きくすると釣鐘型に近づく — これが正規近似(目安 np≧5 かつ n(1−p)≧5)。p を小さく λ=np を一定に保つとポアソン分布に収束する(少数の法則)。

ポアソン分布 Po(λ)

$$P(X=k)=\frac{e^{-\lambda}\lambda^k}{k!}, \qquad E[X]=V[X]=\lambda$$

単位時間あたりの稀な事象の回数(事故件数、来客数)。期待値と分散が等しいのが特徴で、これが成り立たないデータ(過分散)には負の二項分布などを使う。再生性あり:Po(λ₁)+Po(λ₂)~Po(λ₁+λ₂)。

χ²・t・F — 正規分布から派生する3兄弟

分布作り方主な用途
χ²(k)独立な標準正規 Z₁,…,Z_k の二乗和 ΣZᵢ²分散の推定・適合度検定・独立性検定
t(ν)Z / √(W/ν) (Z~N(0,1), W~χ²(ν) 独立)母分散未知の平均の検定・区間推定
F(d₁,d₂)(W₁/d₁)/(W₂/d₂) (Wᵢ~χ²(dᵢ) 独立)分散の比較・分散分析(ANOVA)
t分布は自由度 ν→∞ で標準正規分布に収束する(ラボで ν を上げて確認)。また t(ν) の二乗は F(1, ν) に従う。E[χ²(k)]=k, V[χ²(k)]=2k は暗記必須。

中心極限定理(CLT: Central Limit Theorem)

$$\bar{X}_n \;\text{の分布} \;\longrightarrow\; N\!\left(\mu,\;\frac{\sigma^2}{n}\right) \qquad (n\to\infty,\ \text{元の分布が何であっても})$$

「どんな分布から取ったサンプルでも、平均を取れば正規分布に近づく」。これが正規分布が至る所に現れる理由であり、標本平均に基づく検定・推定の理論的支柱。標準誤差 σ/√n が n の平方根でしか縮まない点(精度を10倍にするにはデータ100倍)も重要。

X ~ B(400, 0.5) のとき、P(X ≧ 220) を正規近似で求めよ。
解答を見る

E[X]=np=200, V[X]=np(1−p)=100 なので X ≈ N(200, 10²)。
Z = (220−200)/10 = 2.0 より P(X≧220) ≈ P(Z≧2.0) ≈ 0.0228。
(連続修正を使うなら (219.5−200)/10=1.95 で ≈0.0256)
