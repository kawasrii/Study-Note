時系列解析:定常性・自己相関・ARMAモデル

タグ: 時系列 / 定常性 / 自己相関 ACF / AR / MA / ARMA / ランダムウォーク / 単位根

前提知識: 分散・共分散・相関係数の定義は基礎(自己相関はその「自分の過去との」版)、分散の演算規則は確率と確率変数にあります。

全体マップ — 時系列解析の体系

時系列データは「観測が独立」というこれまでの前提が崩れた世界。過去との相関(自己相関)をどうモデル化するかが主題で、体系は「定常か非定常か」で大きく分かれる。

  • 時系列解析(順序のあるデータの分析)
    • 記述・分解 — まず眺めて成分に分ける
      • トレンド・季節・不規則成分への分解、移動平均平滑化 — 未収載
    • 定常過程のモデル — 平均・分散・自己相関が時間によらない
      • ホワイトノイズ — 無相関な雑音。全モデルの材料
      • ARモデル(自己回帰) — 自分の過去に回帰する。ACFは指数減衰
      • MAモデル(移動平均) — 過去のノイズの合成。ACFは有限で切断
      • ARMA(p,q) — ARとMAの合成 — 解説のみ
    • 非定常過程 — 平均や分散が時間とともに変わる
      • ランダムウォーク — AR(1)のφ=1(単位根)。分散が発散
      • ARIMA(差分をとって定常化)・季節調整 — 解説のみ
    • 状態空間モデル・スペクトル解析 — 未収載

定常性 — 時系列版「同じ土俵」の条件

(弱)定常とは、①平均が一定、②分散が一定、③自己共分散がラグ k だけで決まる(時点によらない)こと。定常でないと「標本平均や標本ACFが何を推定しているのか」が定まらず、推測の土台が崩れる。だから実務の第一歩は定常化(差分・対数変換・トレンド除去)になる。

AR(1) が定常になる条件 |φ|<1 は最頻出。定常時の性質: E[xₜ]=0、V[xₜ]=σ²/(1−φ²)、自己相関 ρₖ=φᵏ。φ→1 の極限がランダムウォークで、V[xₜ]=tσ² と時間に比例して発散する(=単位根、非定常)。

自己相関関数 ACF — 時系列の指紋

$$\rho_k = \frac{\mathrm{Cov}(x_t,\,x_{t+k})}{V(x_t)}, \qquad \text{標本ACF: } r_k = \frac{\sum_t (x_t-\bar{x})(x_{t+k}-\bar{x})}{\sum_t (x_t-\bar{x})^2}$$

ラグ k 離れた観測同士の相関。モデルによって特徴的な形になるため、ACFの形からモデルを同定するのが古典的手順:AR(p)は指数的に減衰(切れない)、MA(q)はラグ q で切断。ARを見分けるには偏自己相関 PACF(AR(p)は ラグ p で切断)を併用する。

時系列シミュレータ — モデルの「見た目」と「ACFの指紋」を対応させる
下段: 標本ACF(棒)と理論ACF(緑点)。灰帯は「無相関なら95%がこの中」の目安 ±2/√T。

試してほしい操作:AR(1)で φ を 0 → 0.9 へ → 系列が「粘る」ようになり、ACFの減衰が遅くなる。φ を負に → 系列がジグザグし、ACFが正負交互に。MA(1)に切替 → ACFがラグ1だけ立って以降ピタッと消える(切断) — ARとの決定的な違い。ランダムウォーク → 平均に戻らず漂流し、ACFがいつまでも高い(非定常のサイン)。「再生成」を押すたび別の実現値が出る — 同じモデルでも見た目は毎回違うが、ACFの形は保たれることを確認。

🔧 実装メモ(このラボが実際に計算していること)
  • 乱数: Box–Muller法による標準正規乱数(σ=1固定)。シードなし(再生成ごとに異なる実現値)
  • 系列生成: T=150点。AR(1)は初期値0からバーンイン100点を捨てて定常分布に慣らす。MA(1)は xₜ=εₜ+θεₜ₋₁ をそのまま、ランダムウォークは累積和、ホワイトノイズは εₜ
  • 標本ACF: rₖ = Σₜ(xₜ−x̄)(xₜ₊ₖ−x̄) / Σₜ(xₜ−x̄)²(分母はラグ0、k=1…20)
  • 理論ACF(緑点): AR(1)は φᵏ、MA(1)は ρ₁=θ/(1+θ²)・ρ₂以降=0、ホワイトノイズは全て0。ランダムウォークは定常でなく理論ACFが定義できないため非表示
  • 灰帯: ±2/√T ≈ ±0.163。ホワイトノイズ仮定下での標本ACFの近似95%範囲(Bartlett近似の最簡形)
  • 軸固定: 上段 y∈[−8,8](ランダムウォークははみ出したらクリップ。それ自体が「分散発散」の教材)、下段 y∈[−1,1]

ARモデルとMAモデル — 「粘り」と「残響」

$$\mathrm{AR}(1):\ x_t = \varphi x_{t-1} + \varepsilon_t \qquad \mathrm{MA}(1):\ x_t = \varepsilon_t + \theta\varepsilon_{t-1} \qquad (\varepsilon_t:\ \text{ホワイトノイズ})$$

ARは「今日は昨日の φ 倍+ノイズ」— 過去の値が残るので影響が無限に(指数的に減衰しながら)続く。MAは「今日は今日のノイズ+昨日のノイズの θ 倍」— 過去のノイズが q 期だけ残る残響であり、q+1 期先には完全に消える。この違いがそのまま ACF の「減衰 vs 切断」に対応する。両者を合成した ARMA(p,q)、差分してから当てはめる ARIMA(p,d,q) が実用の標準形。

頻出の引っかけ:①「MA過程は常に定常」(パラメータによらない — ARとの対比で問われる)。②AR(1)の定常条件は |φ|<1 であり、φ=1 は単位根=ランダムウォーク。③ランダムウォークは差分 Δxₜ=xₜ−xₜ₋₁ をとるとホワイトノイズ(定常)になる — 「単位根過程は差分で定常化」はARIMAの d の意味そのもの。
AR(1)過程 xₜ = 0.8xₜ₋₁ + εₜ(V[εₜ]=σ²=9)が定常のとき、V[xₜ] と ρ₂(ラグ2の自己相関)を求めよ。
解答を見る

V[xₜ] = σ²/(1−φ²) = 9/(1−0.64) = 25
ρₖ = φᵏ より ρ₂ = 0.8² = 0.64

ある系列の標本ACFがラグ1で0.45、ラグ2以降はほぼ0(±2/√T の帯の内側)だった。AR(1) と MA(1) のどちらが有力か。
解答を見る

MA(1) が有力。AR(1)なら ρ₂=ρ₁²≈0.20 が帯の外に見えるはず(指数減衰)だが、ラグ2で切断されているのはMAの指紋。なお ρ₁=θ/(1+θ²)=0.45 を解くと θ≈0.63(MA(1)の |ρ₁|≤0.5 という制約も確認ポイント)。
