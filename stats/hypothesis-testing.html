仮説検定と検出力

タグ: 仮説検定 / p値 / 有意水準 / 第一種・第二種の過誤 / 検出力 / サンプルサイズ設計

全体マップ — 検定は「何を・どんな条件で」で選ぶ

まず、検定が統計的推測の体系のどこにあるかから。推測統計は「推定」と「検定」の2本柱で、このページは検定を扱う:

  • 統計的推測(標本から母集団を知る)
    • 推定 — 母数の値を当てにいく
      • 点推定(最尤法・モーメント法)— 準備中
      • 区間推定(信頼区間)— 準備中
    • 検定 — 仮説の真偽を判定する ← このページ
      • パラメトリック検定 — 分布(主に正規)を仮定
        • 平均: Z検定(σ既知)・t検定(σ未知)・対応t・Welch・ANOVA(3群以上)
        • 分散: χ²検定(1標本)・F検定(2標本の比)
        • 比率: 比率のZ検定(正規近似)
      • ノンパラメトリック検定 — 分布を仮定しない
        • 度数データ: 適合度χ²・独立性χ²
        • 順位データ: Wilcoxon・Mann–Whitney — 未収載

検定手法は数十種類あるが、選び方は機械的で、①何を検定するか(平均・分散・比率・分布・独立性)と②前提条件(母分散既知か、対応があるか、正規性を仮定できるか)の2つで決まる。準1級で問われる主要な検定を一覧しておく:

重要なのは、どの検定も骨格は完全に同じということ。「H₀を仮定 → 統計量を計算 → 帰無分布上で棄却域と比較」— 違うのは統計量と分布だけ。このページではその共通骨格を、最も見通しの良いZ検定を教材にして徹底的に可視化する。ここが腹落ちすれば、表のどの検定も同じ絵で理解できる。

検定の骨格 — 背理法の確率版

仮説検定は「帰無仮説 H₀ が正しいと仮定したとき、手元のデータほど極端な結果が出る確率(p値)を計算し、それが小さすぎたら H₀ を疑う」という論法。数学の背理法に似ているが、矛盾ではなく「起こりにくさ」で判断する点が違う。

H₀ は「棄却したい仮説」(差がない、効果がない)、H₁ は「示したい仮説」に置くのが定石。検定は H₀ を棄却することはできても採択(証明)することはできない — 棄却できなかったときの結論は「差があるとは言えない」であって「差がない」ではない。

2種類の過誤と検出力

H₀ が真H₁ が真
H₀ を棄却第一種の過誤 (確率 α)正しい判断 (検出力 1−β)
H₀ を棄却しない正しい判断 (1−α)第二種の過誤 (確率 β)

α(あわてんぼうの過誤:ないのにあると言う)と β(ぼんやりの過誤:あるのに見逃す)はトレードオフ。α を下げれば β は上がる。両方下げる唯一の方法がサンプルサイズ n を増やすこと。下のシミュレータで体感してほしい。

棄却域・過誤・検出力シミュレータ(片側Z検定:H₀: μ=0 vs H₁: μ=δ>0)

青い曲線が「H₀ の下での標本平均の分布」、橙が「H₁ の下での分布」。赤い縦線が臨界値で、その右側が棄却域。赤塗り = α、灰塗り = β。n を増やすと両方の山が痩せて(標準誤差 σ/√n が縮む)、重なりが減り、検出力が伸びる様子が見える。

🔧 実装メモ(このシミュレータが実際に計算していること)
  • 設定: 母標準偏差 σ=1 に規格化した片側Z検定(H₀: μ=0 vs H₁: μ=δ>0)。横軸の単位は「σ何個分」。標本平均の標準誤差 SE=1/√n
  • 臨界値 = z₁₋α × SE。分位点 z₁₋α は正規cdf Φ の二分法(探索区間 [−8,8]、60回反復)で数値的に求める
  • 検出力 = 1 − Φ((臨界値 − δ)/SE)。Φ は Abramowitz–Stegun 7.1.26 の有理近似(絶対誤差 < 7.5×10⁻⁸)
  • 軸: x∈[−3,5] 固定(nを増やすと山が痩せる変化がそのまま見えるように)。y は2つの山の高さに合わせ自動
  • 塗り: 赤 = H₀曲線の棄却域側の面積(α)、灰 = H₁曲線の採択域側の面積(β)。両側検定・t検定版は未実装(骨格は同じ)

p値の正しい読み方

p値は「H₀ が正しいとき、観測値以上に極端なデータが得られる確率」。次はすべて誤読:「H₀ が正しい確率」「効果の大きさ」「再現される確率」。p=0.001 は p=0.04 より効果が大きいことを意味しない — 効果量と信頼区間を併記するのが現代の作法。

n が巨大だと無意味に小さい差でも p<0.05 になる(有意 ≠ 重要)。逆に n が小さいと大きな差でも有意にならない。準1級では「検出力 80% を確保するのに必要な n を求めよ」というサンプルサイズ設計の問題が頻出。

サンプルサイズ設計の公式(片側Z検定)

n = ( (z_α + z_β) · σ / δ )²

z_α, z_β は標準正規分布の上側分位点。α=0.05(z=1.645)、検出力 80%(β=0.2, z=0.842)、効果 δ=0.5σ なら n = ((1.645+0.842)/0.5)² ≈ 24.7 → 25人。効果が半分になると必要な n は4倍になる(δ² に反比例)点に注意。

母標準偏差 σ=10 の母集団で、H₀: μ=50 vs H₁: μ>50 を α=0.05、n=25 で片側Z検定する。真の μ が 54 のときの検出力を求めよ。
解答を見る

標準誤差 = 10/√25 = 2。臨界値 = 50 + 1.645×2 = 53.29。
真の分布 N(54, 2²) の下で P(X̄ > 53.29) = P(Z > (53.29−54)/2) = P(Z > −0.355) ≈ 0.639。
上のシミュレータで α=0.05, δ=0.4σ 相当, n=25 に合わせると同じ値が出る。
