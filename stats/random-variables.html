確率の公理 — すべてのルールの出発点

$$\text{① } P(A)\ge 0 \qquad \text{② } P(\Omega)=1 \qquad \text{③ 排反なら } P(A\cup B)=P(A)+P(B)$$

この3つ(コルモゴロフの公理)から確率の性質はすべて導かれる。例: P(Aᶜ)=1−P(A)、一般の加法定理 P(A∪B)=P(A)+P(B)−P(A∩B)(重複の引き戻し)。条件付き確率 P(A|B)=P(A∩B)/P(B) は「Bが起きた世界に縮めて測り直した確率」、独立はP(A∩B)=P(A)P(B)(Bを知ってもAの確率が変わらない)。

確率変数 — 結果に数を割り当てる

確率変数 X とは「偶然の結果を数値に対応させたもの」(サイコロの目、身長、来客数)。取りうる値が数えられる離散型は確率質量関数 p(x)=P(X=x) で、連続的な連続型は確率密度関数 f(x) で分布を表す。連続型では「1点の確率」は0で、面積が確率: P(a≤X≤b)=∫f(x)dx。どちらも全部足す(積分する)と1になる(公理②)。累積分布関数 F(x)=P(X≤x) は両型共通の表現。

期待値の定義 — 分布の重心

$$E[X]=\sum_x x\,p(x) \;\;(\text{離散型}) \qquad E[X]=\int_{-\infty}^{\infty} x\,f(x)\,dx \;\;(\text{連続型})$$

「値 × その値の出やすさ」の総和。記述統計の平均が「データの重心」だったのと同様に、期待値は確率分布の重心 — 確率を重さと見たときの釣り合い点。μ(ミュー)と書き、母平均とも呼ぶ。関数の期待値も同じ形: E[g(X)]=Σg(x)p(x)(値をg(x)に差し替えるだけ)。

期待値の線形性 E[aX+bY]=aE[X]+bE[Y] はXとYが独立でなくても常に成立する。証明スケッチ(離散): E[aX+bY]=ΣΣ(ax+by)p(x,y)=aΣΣx·p(x,y)+bΣΣy·p(x,y)=aE[X]+bE[Y] — 和を分けて周辺化するだけで、独立性をどこにも使っていないことがポイント。

分散の定義 — 重心からの散らばり

$$V[X] = E\big[(X-\mu)^2\big] \qquad (\mu = E[X])$$

「重心からのずれの2乗」の期待値。単位を戻した √V[X] が標準偏差 σ。定義から実用公式を導出する:

$$V[X] = E[X^2-2\mu X+\mu^2] = E[X^2]-2\mu E[X]+\mu^2 = \boxed{E[X^2]-(E[X])^2}$$

展開して線形性を使うだけの2行だが、この「2乗の期待値 − 期待値の2乗」が分散計算の主力。V[aX+b]=a²V[X](位置ずらしbはばらつきに無関係、スケールaは2乗で効く)も定義から同様に出る。

共分散と相関係数 — 2変数の連動

$$\mathrm{Cov}(X,Y) = E\big[(X-\mu_X)(Y-\mu_Y)\big] = E[XY]-E[X]E[Y], \qquad \rho=\frac{\mathrm{Cov}(X,Y)}{\sigma_X\,\sigma_Y}$$

「両方とも平均より上(下)に出やすい」なら正、「逆に動く」なら負。相関係数 ρ は単位の影響を消した規格化版で −1 ≤ ρ ≤ 1。ρ が測るのは直線的な連動だけ(U字関係は ρ≈0 になりうる)。V[X±Y]=V[X]+V[Y]±2Cov(X,Y) への登場が実用上の主戦場 — 発展は多次元確率分布へ。

期待値=重心ラボ — 3つの値の「重み」を変えて、重心と分散の動きを見る

試してほしい操作:x=2の重みだけ大きく → ▲(期待値)が左へ滑る — 確率の重い方へ重心が寄る。真ん中(x=5)に全部寄せる → 分散がほぼ0。両端(x=2とx=8)に半々にする → 期待値は5のままなのに分散が最大 — 「同じ期待値でもばらつきは全く別の情報」を体感。E[X²]−(E[X])² の各項がどう動くかも readout で追える。

🔧 実装メモ(このラボが実際に計算していること)
  • 3点分布 x∈{2,5,8}。スライダーの重み wᵢ を正規化して確率 pᵢ = wᵢ/Σw とする(重み全0のときは計算しない)
  • 期待値 E[X]=Σxᵢpᵢ、E[X²]=Σxᵢ²pᵢ、分散 V=E[X²]−(E[X])² — このページの定義式をそのまま実装(数値誤差対策で max(0,·))
  • 棒の高さ=確率 pᵢ(y軸は0–1固定)。▲が期待値の位置、横棒が ±1σ の範囲
  • 連続型の重心(密度の釣り合い)の可視化は未実装 — 確率分布ラボのE[X]緑点線で代用可
公正なサイコロの出目 X について、定義から E[X] と V[X] を求めよ。
解答を見る

E[X] = (1+2+3+4+5+6)/6 = 21/6 = 3.5
E[X²] = (1+4+9+16+25+36)/6 = 91/6
V[X] = 91/6 − 3.5² = 91/6 − 49/4 = (182−147)/12 = 35/12 ≈ 2.92

区間 [0,1] の連続一様分布(f(x)=1)について、定義から E[X] と V[X] を求めよ。
解答を見る

E[X] = ∫₀¹ x dx = 1/2
E[X²] = ∫₀¹ x² dx = 1/3
V[X] = 1/3 − (1/2)² = 1/12(一様分布の1/12は暗記級の頻出値)
