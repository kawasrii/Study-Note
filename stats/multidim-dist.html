周辺化と条件付け — 2つの基本操作

$$\text{周辺: } f_X(x)=\int f(x,y)\,dy \qquad \text{条件付き: } f(y\mid x)=\frac{f(x,y)}{f_X(x)}$$

周辺化は「興味のない変数を潰して消す」、条件付けは「値を知った上で残りの分布を見る」。独立 ⇔ f(x,y)=f_X(x)f_Y(y) ⇔ 条件付き分布が条件によらない。離散なら積分が和になるだけで構造は同じ。

多項分布 — 二項分布のk色版

$$P(X_1=n_1,\dots,X_k=n_k) = \frac{n!}{n_1!\cdots n_k!}\;p_1^{n_1}\cdots p_k^{n_k}$$

E[Xᵢ]=npᵢ, V[Xᵢ]=npᵢ(1−pᵢ)(周辺が二項だから)。そしてCov(Xᵢ,Xⱼ)=−npᵢpⱼ と必ず負になる — 合計が n に固定されているので、あるカテゴリが多く出れば他は必然的に減るため。この「制約が負の相関を生む」直感は頻出。

2変量正規分布 — 条件付けると回帰直線が現れる

$$Y\mid X=x \;\sim\; N\!\left(\mu_Y+\rho\frac{\sigma_Y}{\sigma_X}(x-\mu_X),\;\; \sigma_Y^2(1-\rho^2)\right)$$

2変量正規の条件付き分布は再び正規で、条件付き期待値は x の1次式 — これが回帰直線 E[Y|x] の正体(傾き ρσ_Y/σ_X は回帰分析の β̂₁=r·s_y/s_x と対応)。条件付き分散 σ_Y²(1−ρ²) は x によらず一定で、決定係数 R²=ρ² が「説明されて減った分散の割合」であることもここから読める。

2変量正規では「無相関 ⇔ 独立」が成立する(一般の分布では不成立)。「周辺が両方正規でも同時分布が2変量正規とは限らない」という逆向きの引っかけもある。
2変量正規ラボ — 相関 ρ が楕円・回帰直線・条件付き分布をどう動かすか(標準化版)

試してほしい操作:ρ を 0 → 0.95 へ → 円が細い楕円に潰れ、x₀での条件付き帯(緑)がキュッと縮む — 「相関が強いほど、xを知ったときのyの不確実性 σ√(1−ρ²) が減る」。ρ=0 → 回帰直線が水平になり帯は±2のまま(xは何も教えてくれない)。ρ を負に → 楕円と直線が逆向きに。そして最重要:回帰直線(青、傾きρ)は楕円の長軸(傾き1)より必ず寝ている — これが「平均への回帰」の幾何学的正体。x₀を端に動かすと、条件付き平均が x₀ ほど極端でないことが見える。

🔧 実装メモ(このラボが実際に計算していること)
  • 標準化版: μ_X=μ_Y=0, σ_X=σ_Y=1 に固定(一般の場合は平行移動と拡大縮小だけの違い)。共分散行列 [[1,ρ],[ρ,1]]
  • 楕円(等高線): マハラノビス距離 c=1, 2 の等高線。固有値 λ₁=1+ρ, λ₂=1−ρ、固有ベクトル (1,±1)/√2 の閉形式を使い、z(θ)=c(√λ₁cosθ·v₁+√λ₂sinθ·v₂) をパラメータ描画(数値の固有値計算なし)
  • 回帰直線: E[Y|x]=ρx(青)。条件付き分布: x₀での N(ρx₀, 1−ρ²) を、点(条件付き平均)と±2σ帯(緑の縦棒)で表示
  • 長軸(灰点線): 傾き±1(ρの符号による)。回帰直線との開きが「平均への回帰」
  • 軸固定: x, y ∈ [−3.5, 3.5]。散布点のサンプリング表示は未実装(等高線=母集団の姿に絞った)

条件付き期待値と全分散の公式 — 分散の分解

$$E[Y]=E\big[E[Y\mid X]\big] \qquad V[Y]=E\big[V[Y\mid X]\big]+V\big(E[Y\mid X]\big)$$

全分散の公式は「Yのばらつき = グループ内のばらつきの平均 + グループ間のばらつき」という分解。分散分析の平方和分解、混合分布の分散計算、階層モデルなど至る所で再登場する、準1級屈指の重要公式。

サイコロを12回振る。1の目の回数を X、2の目の回数を Y とするとき、Cov(X,Y) と相関係数を求めよ。
解答を見る

多項分布で n=12, p₁=p₂=1/6。Cov(X,Y)=−np₁p₂=−12/36=−1/3。
V[X]=V[Y]=12·(1/6)(5/6)=5/3。ρ=(−1/3)/(5/3)=−0.2。

(X,Y)が2変量正規で μ_X=50, μ_Y=60, σ_X=10, σ_Y=5, ρ=0.8 のとき、X=70 を知った下での Y の条件付き期待値と条件付き標準偏差を求めよ。
解答を見る

E[Y|X=70] = 60 + 0.8×(5/10)×(70−50) = 60+8 = 68
V[Y|X] = 5²(1−0.64) = 9 → 条件付き標準偏差 = 3。
(相関を知ることで標準偏差が5→3に減る = 情報の価値)
