回帰分析:最小二乗法から多項式・ノンパラメトリックまで

全体マップ — 回帰手法はこう分類する

回帰手法は無数にあるが、分類軸は実質2つしかない。①関数の形をどこまで仮定するか(パラメトリック ⇄ ノンパラメトリック)と、②答えを点で出すか分布で出すか(頻度論的な点推定 ⇄ ベイズ)。この2軸で整理すると、初見の手法でも地図上のどこにいるか判定できる。

最小二乗法 — 残差の二乗和を最小にする

単回帰モデル y = β₀ + β₁x + ε では、予測とデータのずれ(残差)の二乗和を最小にするように係数を決める。解は closed form で書ける:

β̂₁ = S_xy / S_xx = Σ(xᵢ−x̄)(yᵢ−ȳ) / Σ(xᵢ−x̄)², β̂₀ = ȳ − β̂₁x̄

つまり傾きは「x と y の共分散 ÷ x の分散」。相関係数 r との関係 β̂₁ = r·(s_y/s_x) も頻出。回帰直線は必ず重心 (x̄, ȳ) を通る。

最小二乗法プレイグラウンド — クリック/タップで点を追加、点をもう一度クリックで削除
点を2つ以上打つと回帰曲線が現れます。

試してほしい操作:「サンプル(曲線+ノイズ)」→ 次数1 → 明らかに合わない(過小適合)。次数を2〜3に → よく合う。次数8に → 曲線が点の間で暴れ、全点を無理やり通ろうとする(過学習)。R²は次数を上げるほど必ず上がるが、調整済みR²はどこかで下がり始める。「外れ値を1点追加」→ たった1点で曲線全体が引きずられる。

🔧 実装メモ(このラボが実際に計算していること)
  • モデル: ŷ = Σₖ βₖuᵏ(k=0…d)。x をそのまま使わず u=(x−5)/5 で [−1,1] に正規化してから当てはめる(高次で uᵏ の桁が揃い、数値的に安定するため)
  • 解法: 正規方程式 (XᵀX)β = Xᵀy を部分ピボット選択付きガウス消去で解く。QR分解などは未使用
  • 次数は自動的に d ≤ n−1 に制限(それ以上は解が不定)。行列が特異に近い場合(|pivot|<10⁻¹²)はフィット中止
  • R² = 1−SSE/SST、調整済みR² = 1−(1−R²)(n−1)/(n−d−1)。n−d−1≦0 のときは非表示
  • 曲線は241点の等間隔グリッドで評価して折れ線描画。軸は 0–10 固定で、はみ出しはクリップ
  • 残差線は各データ点から曲線値への鉛直線
「外れ値に弱い」のは残差を二乗するせいで、大きなずれほど過大にペナルティされるから。実務ではまず散布図を描け、と言われる理由。ロバスト回帰(Huber損失など)はこの弱点への対処。

決定係数 R² と調整済み R²

R² = 1 − SSE/SST (単回帰では R² = r²)

R²=0.7 なら「y のばらつきの70%がモデルで説明できる」。ただし R² は説明変数(や次数)を増やすだけで必ず増える。モデル比較には罰則付きの調整済み R² を使う:

調整済みR² = 1 − (1−R²)·(n−1)/(n−p−1) (p: 説明変数の数=多項式なら次数)

多項式回帰と過学習

y = β₀ + β₁x + β₂x² + … + β_d x^d + ε

x の冪を説明変数に加えたモデル。x² や x³ は非線形でも、係数 β について線形なので「線形モデル」の一種であり、最小二乗法がそのまま使える(準1級の引っかけポイント)。

次数 d はモデルの複雑さのダイヤルになっている。上のラボで見た通り:

d が小さすぎる → データの構造を捉えられない(バイアス大 = 過小適合)。d が大きすぎる → ノイズまで拾い、データが少し変わると推定が激変する(バリアンス大 = 過学習)。このバイアス・バリアンスのトレードオフが予測モデル全般を貫く原理で、次数選択には調整済みR²・AIC・交差検証(クロスバリデーション)を使う。

n 個の点は次数 n−1 の多項式で完全に通過できる(R²=1)。だが外挿・内挿は滅茶苦茶になる。「R²が高い=良いモデル」ではない最も鮮明な反例なので、ラボで点6個+次数8を試して目に焼き付けておくこと。
n=10 のデータに、次数1のモデルで R²=0.88、次数3のモデルで R²=0.90 が得られた。調整済みR²でどちらを選ぶべきか。
解答を見る

次数1: 1 − 0.12×9/8 = 0.865。次数3: 1 − 0.10×9/6 = 0.850。
調整済みR²では次数1(単純なモデル)を選ぶ。R²のわずかな改善は複雑化のコストに見合わない。

重回帰と多重共線性

説明変数が複数のモデル y = β₀ + β₁x₁ + … + β_p x_p + ε。係数 βⱼ の解釈は「他の変数を固定したまま xⱼ を1増やしたときの y の変化」— この「他を固定」が単回帰との決定的な違い。

説明変数同士が強く相関していると(多重共線性)、係数の推定が不安定になり、符号が直感と逆転することさえある。診断には VIF = 1/(1−Rⱼ²) を使い、目安として VIF > 10 で問題視する。

準1級では、回帰係数の t 検定(H₀: βⱼ=0)、回帰の分散分析表(F検定)、残差プロットによるモデル診断(等分散性・正規性・独立性の確認)まで問われる。

ノンパラメトリック回帰 — 形を仮定しない

ここまでのモデルは「直線」「d次多項式」という関数の形を先に決めて係数を推定した(パラメトリック)。ノンパラメトリック回帰は形を仮定せず、「近くのデータ点は似た y をもつ」という仮定だけで滑らかな曲線をデータに語らせる。

カーネル回帰(Nadaraya–Watson)

ŷ(x) = Σᵢ K((x−xᵢ)/h)·yᵢ / Σᵢ K((x−xᵢ)/h)

各点 x での予測を、近い点ほど大きい重みをつけた y の加重平均で作る。K はカーネル(ガウス核など)、h はバンド幅で「どこまでを近所とみなすか」を決める。

k近傍(k-NN)回帰

x に最も近い k 個の点の y の平均を予測値にする、最も素朴な方法。曲線は階段状になる。k がバンド幅の役割を果たす。

LOESS(局所回帰)

各 x の近傍データに重み付きの直線(局所的な回帰直線)を当てはめ、その x での値だけ使う。加重平均であるカーネル回帰より端(境界)での偏りが小さく、散布図の傾向線として実務で広く使われる(ggplotの geom_smooth など)。近傍の広さはスパン(全データの何割を使うか)で指定し、重みには tricube 核 (1−t³)³ が定番。

スプライン回帰

区間の境目(節点、ノット)で滑らかにつながる区分多項式(通常3次)を当てはめる。ノットを増やすほど柔軟になる。平滑化スプラインは全点をノットにした上で、曲がりくねりに罰則をかけて滑らかさを自動調整する:

minimize Σ(yᵢ − f(xᵢ))² + λ∫(f″(x))² dx

λ→∞ で直線(最小二乗直線)に、λ→0 で全点を通る補間になる。λ がここでもバンド幅・次数と同じ「複雑さのダイヤル」であることに注目。

点を3つ以上打つと曲線が現れます。

試してほしい操作:「サンプル(波形+ノイズ)」→ バンド幅 h を最小へ → 曲線がノイズを追いかけてギザギザに(バリアンス大)。h を最大へ → ほぼ平らに潰れて波を捉えられない(バイアス大)。ちょうど良い h が中間にある。「段差+ノイズ」で手法を切り替え → k-NNは階段状、カーネルは段差がなまり、LOESSは端で強い。「最小二乗直線と比較」を入れると、形を仮定するモデルとの違いが一目でわかる。

🔧 実装メモ(このラボが実際に計算していること)
  • カーネル回帰: ガウス核 K(u)=exp(−u²/2) を使用。ŷ(x)=Σwᵢyᵢ/Σwᵢ, wᵢ=K((x−xᵢ)/h)。核の正規化定数は分子分母で相殺されるため省略。Epanechnikov核などの他の核は未実装
  • k-NN回帰: |x−xᵢ| の昇順ソートで近傍 k 点を取り、その y の重みなし単純平均。k はデータ数 n に自動クランプ
  • LOESS: 近傍 m=⌈スパン×n⌉ 点に tricube 重み w=(1−t³)³, t=|x−xᵢ|/(近傍内最大距離) を付け、局所1次(直線)を加重最小二乗で解いてその x の値だけ採用。加重分散が退化した場合は加重平均にフォールバック。本家LOESSにあるロバスト化反復(外れ値の重み引き下げ)は未実装
  • 比較用の「最小二乗直線」はラボ1と同じ多項式フィッタの次数1
  • 曲線は241点グリッドで評価。軸は 0–10 固定
バンド幅 h・近傍数 k・スパン・罰則 λ・多項式の次数 d — 名前は違えど全部「複雑さ(平滑さ)を決めるチューニングパラメータ」であり、どれもバイアス・バリアンスのトレードオフを調整している。選択には交差検証を使うのが原則。この統一的な見方が身につけば、初見の手法でも本質を掴める。

ガウス過程回帰 — 「もっともらしい関数」の分布を推定する

ここまでの手法はどれも「1本の曲線」を答えとして返した。ガウス過程回帰(GPR)は発想が根本から違う。「データに合いそうな関数すべての上の確率分布」を答えとして返す — つまり各 x での予測が点ではなく分布(平均と分散)で出てくる。

出発点は、関数 f そのものに事前分布を置くこと。「近い x では f の値も近い」という滑らかさの信念を、カーネル(共分散関数)で表す:

k(x, x′) = σ_f² · exp( −(x−x′)² / 2ℓ² ) (RBFカーネル。ℓ: 長さスケール)

データ (X, y) で条件付けると、新しい点 x* での予測分布が閉じた式で得られる:

平均 μ* = k*ᵀ(K+σ_n²I)⁻¹y, 分散 σ*² = k(x*,x*) − k*ᵀ(K+σ_n²I)⁻¹k*

ここが最大の特徴で、データが密な場所では分散が小さく(自信あり)、データが無い場所では事前分布に戻って分散が大きくなる(正直に「わからない」と言う)。下のラボの青い帯(平均±2σ、95%信用区間)がその「もっともらしさ」の可視化そのもの。

ガウス過程回帰ラボ — クリック/タップで点を追加、もう一度クリックで削除。帯 = 平均±2σ

試してほしい操作:まず点が無い状態を見る → 帯が全域で太い=「何もわからない」状態(事前分布)。細いサンプルパスは事前分布から引いた「あり得る関数」たち。「サンプル(隙間のあるデータ)」→ 点の近くで帯がキュッと締まり、データの隙間と両端でふくらむ。サンプルパスも点の近くでは束になり、隙間では散らばる — これが「もっともらしさ」の姿。σₙ を上げる → 点を厳密に通らなくなり帯も緩む(データを疑っている)。ℓ を小さく → 曲線が細かく曲がれるようになり、点から少し離れただけで帯が開く。

🔧 実装メモ(このラボが実際に計算していること)
  • カーネル: RBF(二乗指数)のみ k(x,x′)=σ_f²·exp(−(x−x′)²/2ℓ²)。信号分散 σ_f²=6.25(σ_f=2.5)に固定(キャンバスの縦幅 0–10 に事前分布の±2σ帯が収まる値)。Matérn核・周期核などは未実装
  • 平均関数: データの y の標本平均を引いてゼロ平均GPとして計算し、予測時に足し戻す(データなし時は 5 = キャンバス中央)
  • 事後分布は厳密計算: μ*=k*ᵀ(K+σₙ²I)⁻¹y, σ*²=k(x*,x*)−k*ᵀ(K+σₙ²I)⁻¹k*。逆行列は部分ピボット付きガウス消去(O(n³))。数値安定化のため K の対角に 1e-8 のジッターを加算
  • 青帯: 平均±2σ(正規分布なので約95%信用区間)。σ は関数値 f の不確実性で、観測ノイズ σₙ² は帯に含めていない(含めると予測区間になる)
  • サンプルパス: 81点グリッド上の事後共分散行列(対角に 1e-6 ジッター)をCholesky分解し、f=μ+Lz(z は Box–Muller 法による標準正規乱数)で3本生成
  • ℓ・σₙ の周辺尤度最大化(自動調整)は未実装 — スライダーで手動調整する設計(その方が役割を体感できるため)
カーネル回帰(Nadaraya–Watson)と式は似ているが、GPRはベイズ推論なので不確実性が定量的に出るのが本質的な違い。「どこを次に測定・実験すべきか」を分散が教えてくれるため、ベイズ最適化(ハイパーパラメータ探索)や実験計画で主力になっている。弱点は計算量 O(n³)(逆行列)で、大規模データには近似法を使う。ℓ や σₙ 自体は周辺尤度の最大化で自動調整できる — これもベイズ流の利点。
n=10 のデータで x̄=5, ȳ=20, S_xx=40, S_xy=60, S_yy=100 のとき、回帰直線と決定係数を求めよ。
解答を見る

β̂₁ = 60/40 = 1.5, β̂₀ = 20 − 1.5×5 = 12.5 → ŷ = 12.5 + 1.5x
R² = S_xy²/(S_xx·S_yy) = 3600/4000 = 0.9(r = 0.949)
