質的変数の回帰と一般化線形モデル(GLM)

タグ: ダミー変数 / ロジスティック回帰 / オッズ / オッズ比 / ロジット / ポアソン回帰 / リンク関数 / 一般化線形モデル

フェーズ2対応済み: ロジスティック曲線ラボ・推定と評価(逸脱度)の節つき。

前提知識: 回帰分析(重回帰と係数の解釈)、推定(最尤法 — GLMの係数推定はこれ)、オッズ・オッズ比は分割表の解析にも登場します。

全体マップ — この分野の体系

通常の回帰は「連続な y を正規ノイズで」という前提だった。y が 0/1 や回数のとき、その前提を外して回帰の枠組みを拡張したのが一般化線形モデル(GLM: Generalized Linear Model)。「質的な変数が説明側に来る場合(ダミー変数)」と「応答側に来る場合(ロジスティック回帰など)」を区別して整理する。

  • 質的変数と回帰・GLM
    • 説明変数が質的
      • ダミー変数(0/1)の導入と係数の解釈
      • 交互作用項・カテゴリ数が多い場合 — 触りのみ
    • 応答変数が質的・非正規(GLM)
      • 2値応答: ロジスティック回帰 — 準1級最頻出
      • 計数応答: ポアソン回帰
      • プロビットモデル・順序ロジット — 未収載
    • GLMの統一的枠組み
      • 3要素: 分布(指数型分布族)+線形予測子+リンク関数
      • 推定は最尤法(反復計算)。評価は逸脱度・AIC(モデル選択)

ダミー変数 — カテゴリを回帰に乗せる

$$y = \beta_0+\beta_1 x+\beta_2 D+\varepsilon \qquad (D=1:\ \text{処置群},\; D=0:\ \text{対照群})$$

β₂ は「他の条件が同じとき、D=1 グループは D=0 より平均で β₂ だけ高い」という切片の差。カテゴリが k 種なら、ダミーは k−1 本(基準カテゴリを1つ決める) — k 本入れると完全な多重共線性で推定不能になる(ダミー変数トラップ)。交互作用項 D×x を足せば「傾きの差」も表せる。

ロジスティック回帰 — 確率を回帰する正しい方法

$$\ln\frac{p}{1-p} = \beta_0+\beta_1 x \quad\Longleftrightarrow\quad p = \frac{1}{1+e^{-(\beta_0+\beta_1 x)}}$$

確率 p をそのまま直線で当てると 0–1 の範囲を飛び出す。そこで p をオッズ p/(1−p)(起こる:起こらない の比)にし、対数を取ったロジットを線形にする。すると p は滑らかなS字(シグモイド)曲線になり、必ず 0–1 に収まる。

係数の解釈が最頻出: x が1増えるとロジットが β₁ 増える = オッズが e^β₁ 倍になる(オッズ比)。e^β₁=1.5 なら「x が1増えるとオッズが1.5倍」。確率 p そのものが1.5倍になるのではない点に注意。推定は最尤法(閉じた式はなく反復計算)、係数の検定はWald検定 z=β̂/SE。
ロジスティック曲線ラボ — 係数がS字とオッズをどう動かすか

試してほしい操作:β₁ を大きく → S字が急峻になる(判別がシャープに)。β₁ を 0 に → 水平線(x が結果に無関係)。β₁ を負に → S字が反転(x が増えるほど確率低下)。β₀ を動かす → 曲線が左右に平行移動し、50%になる位置 x₅₀ = −β₀/β₁ が滑る。readoutのオッズ比 e^β₁ は β₁ だけで決まり、β₀ には依存しないことも確認。

🔧 実装メモ(このラボが実際に計算していること)
  • 曲線は p(x)=1/(1+e^{−(β₀+β₁x)}) を x∈[0,10](300点)で評価。y軸は[0,1]固定
  • 50%点 x₅₀=−β₀/β₁(β₁≠0のとき)に縦点線。オッズ比 e^{β₁} と p(5) をreadout表示
  • データからの推定(IRLS)・信頼帯は未実装 — 係数→形の対応を体感する決定的可視化に絞っている

推定と評価 — 最尤法・逸脱度・AIC

GLMの係数は最尤法で推定するが、線形回帰と違い閉じた式がなく、反復重み付き最小二乗(IRLS: Iteratively Reweighted Least Squares)で数値的に解く(統計ソフトが中でやっていること)。当てはまりの評価には残差平方和の代わりに逸脱度(deviance)を使う:

$$D = -2\big(\ln L_{\text{モデル}} - \ln L_{\text{飽和モデル}}\big) \qquad (\text{小さいほど良い。線形回帰のSSEに相当})$$

ネストしたモデルの比較は逸脱度の差(尤度比検定、χ²分布)、ネストしない比較はAIC — 線形回帰で学んだ道具立てが尤度の言葉でそのまま再現される。

ポアソン回帰 — 回数データの回帰

$$\ln\lambda = \beta_0+\beta_1 x \qquad \big(y \sim \mathrm{Po}(\lambda)\big)$$

事故件数・来店回数など非負の計数データ用。対数リンクなので係数は乗法的に効く:x が1増えると期待回数が e^β₁ 倍。観測期間が違うデータはオフセット項 ln(期間) を加えて率を揃える。過分散(分散>平均)のときはポアソンの前提が崩れており、負の二項回帰などを検討 — という論点が頻出。

GLMの3要素 — 統一フレーム

①応答の分布(指数型分布族: 正規・二項・ポアソン・ガンマ…)、②線形予測子 η=xβ、③リンク関数 g(μ)=η。この3点セットを選ぶだけで回帰が組める、というのがGLMの見方。線形回帰もロジスティックもポアソン回帰も同じ機械の設定違いにすぎない — この統一観が身につくと暗記が激減する。

ロジスティック回帰で合格(y=1)を勉強時間 x で説明したところ β̂₁=0.8 だった。勉強時間が1時間増えると何がどう変わるか。オッズ比で答えよ。
解答を見る

オッズ比 = e^0.8 ≈ 2.23。勉強時間が1時間増えると合格オッズが約2.23倍になる。
(確率が2.23倍になるのではない。例えば p=0.5(オッズ1)からなら新オッズ2.23 → p=2.23/3.23≈0.69)

β₀=−3, β₁=0.5 のロジスティックモデルで、x=8 のときの予測確率を求めよ。
解答を見る

ロジット = −3+0.5×8 = 1。p = 1/(1+e^{−1}) ≈ 0.731。

確率 p=0.2 のとき、オッズとロジット(対数オッズ)を求めよ。また p=0.8 のときのロジットとの関係を述べよ。
解答を見る

オッズ = 0.2/0.8 = 0.25、ロジット = ln 0.25 ≈ −1.386。
p=0.8 のオッズは 4、ロジットは +1.386 — p と 1−p でロジットは符号が反転するだけ(ロジットは p=0.5 を中心に対称)。

ロジスティックモデルで予測確率がちょうど50%になる x の値を β₀, β₁ で表せ。β₀=−4, β₁=0.8 のときの値は?
解答を見る

p=0.5 ⇔ ロジット=0 ⇔ β₀+β₁x=0 → x₅₀ = −β₀/β₁。
β₀=−4, β₁=0.8 なら x₅₀ = 4/0.8 = 5(上のラボの初期設定で縦点線が x=5 に立つ)。
