全体マップ — この分野の体系

「3群以上の平均を比べる」道具が分散分析(ANOVA)、「比較が正しくできるようにデータの取り方を設計する」のが実験計画法。後者が先にあって前者が活きる(計画が悪いと解析では救えない)という関係を押さえる。

フィッシャーの3原則 — 比較を成立させる設計

①反復(同条件を複数回 → 誤差分散が推定できる)、②無作為化(処理の割付をランダムに → 系統誤差を偶然誤差に転化。因果推論の土台でもある)、③局所管理(似た条件のブロック内で比較 → 既知のばらつき要因を除去)。どの操作がどの原則かを判定させる問題が定番。

一元配置分散分析 — ばらつきを2つに割る

SST(全変動) = SSB(群間変動) + SSW(群内変動)

「群平均が互いに離れているか」を「群内の自然なばらつき」と比べるのが本質。H₀: μ₁=…=μ_a のもとで F=MSB/MSW ~ F(a−1, n−a) となり、Fが大きければ棄却。t検定を繰り返さない理由(検定の多重性: 5%の検定を繰り返すと全体の第一種過誤率が膨らむ)もセットで頻出。この分解は全分散の公式(群内平均+群間)の標本版。

二元配置と交互作用 — 「組み合わせの妙」を検出する

SST = SS_A + SS_B + SS_{A×B} + SSE

因子が2つ(例: 肥料×品種)のとき、各因子の主効果に加えて交互作用(片方の効果がもう片方の水準によって変わる)を分離できる。交互作用図で線が平行 → 交互作用なし、交差・非平行 → あり。交互作用が有意なら主効果の単独解釈は危険、が最重要の読み方。交互作用を推定するには各セルに反復が必要。

前提は「各群の母集団が正規で等分散」。分散分析は名前に反して「平均の差」の検定であることも引っかけの定番。乱塊法はブロック(局所管理)を要因として取り込んだ設計で、ブロック間変動を誤差から除くため検出力が上がる。
3群(各群5個、n=15)の一元配置で SST=100、SSB=40 だった。分散分析表を完成させ、F値を求めよ(F₀.₀₅(2,12)=3.89)。
解答を見る

SSW=100−40=60。自由度: 群間 a−1=2, 群内 n−a=12。
MSB=40/2=20, MSW=60/12=5 → F=20/5=4.0 > 3.89 → 有意水準5%で棄却(群平均に差があるといえる)。

「畑を土質の似た4ブロックに分け、各ブロック内で3種の肥料をランダムに割り付けた」— フィッシャーの3原則のどれが使われているか。
解答を見る

局所管理(ブロック化)と無作為化(ブロック内のランダム割付)。各肥料が複数ブロックに現れるので反復も満たす — 3原則すべてを使った乱塊法の典型例。
