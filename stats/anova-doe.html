分散分析と実験計画法

タグ: 分散分析 ANOVA / 平方和の分解 / F検定 / 交互作用 / フィッシャーの3原則 / 乱塊法 / 直交表

フェーズ2対応済み: 平方和分解が動くラボ・多重比較の節つき。

前提知識: F分布(χ²の比)は確率分布ラボ、検定の共通骨格(棄却域・p値)は仮説検定、平方和分解の親玉「全分散の公式」は多次元確率分布にあります。

全体マップ — この分野の体系

「3群以上の平均を比べる」道具が分散分析(ANOVA)、「比較が正しくできるようにデータの取り方を設計する」のが実験計画法。後者が先にあって前者が活きる(計画が悪いと解析では救えない)という関係を押さえる。

フィッシャーの3原則 — 比較を成立させる設計

①反復(同条件を複数回 → 誤差分散が推定できる)、②無作為化(処理の割付をランダムに → 系統誤差を偶然誤差に転化。因果推論の土台でもある)、③局所管理(似た条件のブロック内で比較 → 既知のばらつき要因を除去)。どの操作がどの原則かを判定させる問題が定番。

一元配置分散分析 — ばらつきを2つに割る

$$\mathrm{SST}\;(\text{全変動}) = \mathrm{SSB}\;(\text{群間変動}) + \mathrm{SSW}\;(\text{群内変動})$$

「群平均が互いに離れているか」を「群内の自然なばらつき」と比べるのが本質。H₀: μ₁=…=μ_a のもとで F=MSB/MSW ~ F(a−1, n−a) となり、Fが大きければ棄却。t検定を繰り返さない理由(検定の多重性: 5%の検定を繰り返すと全体の第一種過誤率が膨らむ)もセットで頻出。この分解は全分散の公式(群内平均+群間)の標本版。

平方和分解ラボ — 3群(A・B・C)の帯にクリック/タップで点を置く(もう一度で削除)
各群に2点以上置くと分散分析表が動きます。

試してほしい操作:「差あり」サンプル → F が大きく p<0.05。「差なし」→ F は小さい。差ありの状態から各群の点を上下に大きく散らす → 群平均の差(色つき横線の開き)は同じなのに F が下がって有意でなくなる — 検定は「信号(群間) ÷ 雑音(群内)」の比だという本質。1つの群を丸ごと上に動かすと SSB だけが伸びるのも確認。

🔧 実装メモ(このラボが実際に計算していること)
  • 群は横位置で固定(A: x∈[0.7,3.3] / B: [3.7,6.3] / C: [6.7,9.3])。縦位置 y∈[0,10] が測定値。クリックした帯の群に点が入る
  • SSB=Σ n_h(ȳ_h−ȳ)²、SSW=ΣΣ(y−ȳ_h)²、F=(SSB/(a−1))/(SSW/(n−a))。a=3固定(点のない群は除いて自由度を調整)
  • p値はF分布pdf(共通エンジンのfPdf、Lanczos近似ベース)の数値積分: 台形則で[0,F]を2000分割してCDFを作り p=1−CDF。積分上限誤差を抑えるためF>50はp≈0と表示
  • 横線: 各群平均(群色)と全体平均(灰)。灰線と群線の距離がSSBの素、点と群線の距離がSSWの素
  • 等分散・正規性の診断、二元配置、多重比較の実装はなし(解説のみ)

二元配置と交互作用 — 「組み合わせの妙」を検出する

$$\mathrm{SST} = \mathrm{SS}_A + \mathrm{SS}_B + \mathrm{SS}_{A\times B} + \mathrm{SSE}$$

因子が2つ(例: 肥料×品種)のとき、各因子の主効果に加えて交互作用(片方の効果がもう片方の水準によって変わる)を分離できる。交互作用図で線が平行 → 交互作用なし、交差・非平行 → あり。交互作用が有意なら主効果の単独解釈は危険、が最重要の読み方。交互作用を推定するには各セルに反復が必要。

前提は「各群の母集団が正規で等分散」。分散分析は名前に反して「平均の差」の検定であることも引っかけの定番。乱塊法はブロック(局所管理)を要因として取り込んだ設計で、ブロック間変動を誤差から除くため検出力が上がる。

多重比較 — 「どの群とどの群が違うか」を正しく調べる

なぜ必要か: ANOVAのFが有意でも、分かるのは「どこかに差がある」だけ。ではどのペアか?と全ペアにt検定を繰り返すと、1回5%の誤検出が積み重なって全体の第一種過誤率が膨張する(検定の多重性)。m回の独立な検定なら全体の過誤率は最大 1−(1−α)^m — 6回で約26%にもなる。

$$\text{Bonferroni補正: 各検定を } \alpha' = \frac{\alpha}{m} \text{ で行えば、全体の過誤率} \le \alpha$$

Bonferroniは単純で常に使えるが保守的(検出力が落ちる)。全ペア比較専用にはTukeyのHSD(スチューデント化された範囲分布を使う)がより効率的、というのが定番の使い分け。

4群の全ペア(m=6)に有意水準5%のt検定を繰り返すとき、(a) 全体でどこかのペアが誤検出される確率の上限(独立と仮定)を求めよ。(b) Bonferroni補正後の各検定の有意水準は?
解答を見る

(a) 1−0.95⁶ ≈ 0.265(約26.5%) — 「5%のつもりが4分の1」。
(b) α′ = 0.05/6 ≈ 0.0083。

3群(各群5個、n=15)の一元配置で SST=100、SSB=40 だった。分散分析表を完成させ、F値を求めよ(F₀.₀₅(2,12)=3.89)。
解答を見る

SSW=100−40=60。自由度: 群間 a−1=2, 群内 n−a=12。
MSB=40/2=20, MSW=60/12=5 → F=20/5=4.0 > 3.89 → 有意水準5%で棄却(群平均に差があるといえる)。

「畑を土質の似た4ブロックに分け、各ブロック内で3種の肥料をランダムに割り付けた」— フィッシャーの3原則のどれが使われているか。
解答を見る

局所管理(ブロック化)と無作為化(ブロック内のランダム割付)。各肥料が複数ブロックに現れるので反復も満たす — 3原則すべてを使った乱塊法の典型例。
