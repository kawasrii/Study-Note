全体マップ — この分野の体系

数式で解けない問題を「乱数で実験して」解く技術群。①欲しい分布の乱数を作る → ②大量に流して期待値・確率を近似する(モンテカルロ) → ③手元のデータ自体を乱数の源にする(リサンプリング)の3層構造。

逆関数法 — 一様乱数を任意の分布に変換する

$$U \sim \mathrm{U}(0,1)\ \text{のとき}\quad X = F^{-1}(U)\ \text{は分布関数}\ F\ \text{に従う}$$

根拠は P(F⁻¹(U)≤x) = P(U≤F(x)) = F(x) の1行。累積分布関数の逆関数が書ける分布(指数・コーシー・ワイブル等)ではこれが最速。書けない正規分布にはボックス=ミュラー法(一様乱数2つ → 独立な標準正規2つ:このサイトの全ラボの正規乱数はこれ)、密度しか分からない分布には棄却法(覆いからサンプルして受理/棄却)を使う。

モンテカルロ法 — 「サイコロで積分する」

$$E[g(X)] \approx \frac{1}{n}\sum_{i=1}^{n} g(x_i), \qquad \text{標準誤差} = \frac{\sigma_g}{\sqrt{n}}$$

大数の法則がそのまま計算アルゴリズムになったもの。重要なのは精度のスケーリング:誤差は 1/√n — 桁を1つ上げるのに試行が100倍要る。一方で誤差率が次元によらないため、高次元積分では数値積分の格子法より圧倒的に有利になる。

ブートストラップ — 標本を母集団の身代わりにする

手順:①手元の標本(サイズn)から復元抽出でn個取り直した「ブートストラップ標本」をB回(例: 2000回)作る → ②各標本で統計量 θ̂*を計算 → ③θ̂*のばらつきを標準誤差の推定に、分位点を信頼区間(パーセンタイル法)に使う。中央値や相関係数など、標準誤差の理論式が難しい統計量でも機械的に精度評価できるのが革命的な点。

注意点が頻出:①ブートストラップは「n が小さすぎる」問題を救わない(標本が母集団を代表していることが前提)。②時系列など独立でないデータには素朴な復元抽出は不可(ブロック・ブートストラップ等が必要)。③並べ替え検定との違い — 並べ替えはH₀を課してラベルを混ぜる(検定用)、ブートストラップは分布ごと再現する(推定精度用)。
一様乱数 U から、指数分布 Exp(λ) に従う乱数を逆関数法で作る式を導け。
解答を見る

F(x)=1−e^{−λx} を u について解く: u=1−e^{−λx} → x=−ln(1−u)/λ。
X = −ln(1−U)/λ(1−U も一様なので X=−ln U/λ でも同じ)。

モンテカルロ法で確率 p≈0.5 を標準誤差 0.005 以下で推定したい。必要な試行回数はおよそいくらか。
解答を見る

SE=√(p(1−p)/n) ≤ 0.005 → n ≥ 0.25/0.005² = 10,000回。
さらに10倍の精度(0.0005)には100万回 — 1/√n の遅さの体感値として覚えておく。
